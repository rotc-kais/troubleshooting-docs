# demo 演示环境gitlab异常

升级时磁盘空间不足 镜像仓库异常

## Cause
- 磁盘空间不足导致升级失败

## Resolution
- 释放磁盘空间
- 重置镜像仓库
- 重新上传节点缓存镜像
