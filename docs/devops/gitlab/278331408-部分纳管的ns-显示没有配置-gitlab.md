# 部分纳管的ns 显示没有配置 gitlab

部分纳管的ns显示没有配置gitlab

## Cause
- gitlab已存在的test-gpu group可见性设置为private，导致无法创建subgroup

## Resolution
- 将test-gpu group可见性改为public
- 删除已存在的test-gpu group让系统自动创建
