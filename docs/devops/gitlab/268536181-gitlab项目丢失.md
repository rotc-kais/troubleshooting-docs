# gitlab项目丢失

所有仓库在用户视图中无法访问 仓库检查（repocheck）全部失败

## Cause
- 仓库检查机制异常触发全局访问限制
- 存储路径变更后可能存在的配置残留

## Resolution
- 1. 访问 Admin 控制台：Menu > {admin} Admin
- 2. 导航至仓库设置：Settings > Repository (/admin/application_settings/repository)
- 3. 展开 Repository maintenance 区块
- 4. 执行 Clear all repository checks 操作
