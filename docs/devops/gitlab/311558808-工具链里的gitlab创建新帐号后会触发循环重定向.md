# 工具链里的gitlab创建新帐号后，会触发循环重定向

新用户登录后跳转至密码重置页面 出现循环重定向错误 ERR_TOO_MANY_REDIRECTS 通过Impersonate修改密码后仍无法正常登录

## Cause
- 创建用户时使用了特殊格式的临时邮箱 temp-email-for-oauth-*

## Resolution
- 将用户邮箱修改为有效格式（如 用户名@ztccloud.com.cn）
- 参考官方issue 427976处理
