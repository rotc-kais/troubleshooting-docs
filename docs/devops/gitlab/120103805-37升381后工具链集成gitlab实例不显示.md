# 3.7升3.8.1后工具链集成gitlab实例不显示

工具链集成中的GitLab实例信息不显示

## Cause
- 集成API地址末尾包含多余斜杠/导致地址匹配失败

## Resolution
- 1. 进入集成配置界面
2. 编辑GitLab集成
3. 移除API地址和访问地址末尾的/
4. 更新保存配置
