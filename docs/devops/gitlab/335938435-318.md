# 3.18

prdb phase 为 Provisioning 所有 pod 运行正常

## Cause
- base-operator 在找不到 alauda-operators 镜像时触发卸载
- cluster-transformer 的 imageExists 接口在 3.18 版本存在判断逻辑缺陷

## Resolution
- 删除 alauda-operators 的 finalizer 并等待重建
