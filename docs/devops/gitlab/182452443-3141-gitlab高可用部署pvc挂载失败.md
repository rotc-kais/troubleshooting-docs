# 3.14.1 gitlab高可用部署pvc挂载失败

pgsql pod启动不起来 pvc挂载失败

## Cause
- 使用了topolvm块存储导致访问模式不兼容
- 存储类不支持多节点读写(RWX)

## Resolution
- 更换支持多节点读写的存储类
- 手动创建符合部署文档要求的PVC
