# GitLab 打开 SSO 问题

通过 dex 登录遇到 500 错误码

## Cause
- 平台访问地址未使用域名形式
- 域名证书不合法

## Resolution
- 设置平台的访问地址为域名模式
- 配置合法的域名证书
- 更新 gitlab 所在集群的 cluster 资源 annotation: cpaas.io/platform-url: <https://publicIP:443>
