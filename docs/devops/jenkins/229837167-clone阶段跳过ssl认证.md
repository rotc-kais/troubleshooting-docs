# clone阶段跳过ssl认证

server certificate verification failed CAfile none CRLfile none

## Cause
- gitlab使用https自签或非法证书导致ssl验证失败

## Resolution
- 在构建节点base pod中设置环境变量GIT_SSL_NO_VERIFY=true
- 重启jenkins服务
- 在流水线clone阶段添加git参数：git config --global http.sslVerify "false"
