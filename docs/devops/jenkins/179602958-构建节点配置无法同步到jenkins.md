# 构建节点配置无法同步到jenkins

构建节点配置无法同步到Jenkins Jenkins日志中报错关于响应reload-config接口的报错 jenkins-sc-config未正常运行

## Cause
- 用户自行安装Publish over ssh插件导致snakeYaml插件版本过高

## Resolution
- 备份故障Jenkins数据目录
- 删除$JENKINS_HOME/plugin/*
- 从正常Jenkins拷贝plugin目录到故障实例
- 重启Jenkins
