# buildah构建缓慢

buildah构建镜像过程缓慢 copy操作执行缓慢

## Cause
- 构建目录使用fuse overlay文件系统

## Resolution

## [workaround]
- 修改Jenkins的ConfigMap(jenkins-xxxxx-mount)，将buildah容器名称从默认'buildah'改为其他名称(如buildah123)，触发退级使用docker build

## [Related Information]
- Environment: 3.6.x
- buildah容器名称
- jenkins-xxxxx-mount
- fuse overlay文件系统
