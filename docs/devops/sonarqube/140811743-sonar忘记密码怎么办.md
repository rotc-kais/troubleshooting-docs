# sonar忘记密码怎么办

无法登录SonarQube管理员账户 忘记管理员密码

## Cause
- 管理员账户密码丢失或未知

## Resolution
- 通过kubectl进入sonar-postgresql容器
- psql命令连接sonarDB数据库
- 执行SQL更新users表密码字段
- 使用admin/admin临时凭证登录
- 强制重置新密码

## [workaround]

## [Related Information]
- Environment: TKE 3.8 以上
- sonar-postgresql
- sonarUser
- sonarDB
- users表
- crypted_password字段
- PBKDF2哈希算法
