# 应用拉取镜像失败

应用更新镜像版本后拉取镜像失败 kubectl describe pod 显示 unauthorized to access repository

## Cause
- 应用中配置的 imagePullSecret 账号无法正常登录镜像仓库

## Resolution
- 通过页面更新对应的保密字典，将账号密码更新正确
