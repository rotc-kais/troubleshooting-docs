# 应用镜像拉取失败

镜像拉取失败，报400错误 业务镜像下载提示401未授权

## Cause
- containerd配置中http协议优先级高于https协议
- 镜像凭据中的密码不正确
- 容灾环境未同步镜像到两边仓库

## Resolution
- 修改containerd配置将https协议调整为第一个
- 更新镜像凭据中的正确密码
- 在容灾环境两边镜像仓库同时上传所需镜像
