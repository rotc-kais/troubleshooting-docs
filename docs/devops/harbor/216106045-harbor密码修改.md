# harbor密码修改

无法使用修改后的密码登录Harbor

## Cause
- 忘记修改后的Harbor管理员密码

## Resolution
- 通过kubectl exec进入Harbor数据库Pod
- 执行psql -U postgres -d postgres -h 127.0.0.1 -p 5432进入PostgreSQL
- 切换到registry数据库执行UPDATE harbor_user SET password和salt字段
- 使用Python脚本生成新密码哈希值

## [workaround]
- 新建Harbor实例获取新生成的password和salt值
