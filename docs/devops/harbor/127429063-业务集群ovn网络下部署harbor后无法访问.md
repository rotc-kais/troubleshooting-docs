# 业务集群ovn网络下，部署harbor后无法访问

部署harbor后无法访问 节点curl harbor svc不通 节点curl harbor pod-ip:8080不通

## Cause
- harbor所在命名空间设置了networkpolicy，禁止非harbor命名空间的流量入站

## Resolution
- 删除该networkpolicy
