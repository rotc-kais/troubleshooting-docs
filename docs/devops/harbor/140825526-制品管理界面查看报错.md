# 制品管理界面查看报错

Http failure response for https://10.156.140.184/integrations/v1alpha1/namespaces/iip/integrations/cesiharbor-8gzgap/projects/sinochem.com: 404 OK

## Cause
- Harbor项目被删除后未解除平台分配关系

## Resolution
- 在Harbor平台重新创建已删除的项目
- 在工具链管理中解除该项目的平台分配
- 删除工具链管理中的项目配置
