# 厦门

failed to garbage collect error enumerating blobs filesystem unsupported digest algorithm

## Cause
- Harbor 1.8旧版本无法识别新版本Docker产生的镜像摘要算法

## Resolution
- 升级Harbor版本至兼容新Docker算法的版本
