# 推理服务报错拉取镜像509

拉取镜像报错509 没有创建任何Pod

## Cause
- Knative未配置跳过指定镜像仓库的标签解析

## Resolution
- 修改knative-serving命名空间下的config-deployment ConfigMap，添加registries-skipping-tag-resolving: <仓库>配置
