# 3.12.1

Redis哨兵节点未Ready Harbor相关资源标签不一致

## Cause
- app.kubernetes.io/managed-by标签被修改为redis-operator
- 数据服务Redis纳管机制与Harbor组件冲突

## Resolution
- 删除harbor-redis相关statefulset/deploy/service资源
