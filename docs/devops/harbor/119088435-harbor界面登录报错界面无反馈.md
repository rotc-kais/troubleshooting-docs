# harbor界面登录报错界面无反馈

harbor界面登录报错界面无反馈 服务状态正常但无法正常访问

## Cause
- 多个harbor实例使用相同的ingress url导致冲突

## Resolution
- 重新部署存在冲突的harbor实例并更改对外暴露地址
