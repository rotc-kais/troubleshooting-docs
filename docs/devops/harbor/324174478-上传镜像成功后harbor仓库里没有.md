# 上传镜像成功后，harbor仓库里没有

上传镜像成功后，Harbor仓库镜像列表未显示新上传的tag 多次尝试后仍无法在默认分页找到镜像

## Cause
- 新打的tag覆盖了旧的tag，导致旧tag被移动到分页的下一页

## Resolution
- 检查镜像列表分页的下一页确认是否存在
- 确认是否将新tag覆盖到了旧tag上
