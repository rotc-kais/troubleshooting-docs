# 平台部署异常

平台部署卡在安装镜像仓库阶段 镜像仓库pod处于crash状态 端口对应的进程kill后重启pod无效

## Cause
- 宿主机已存在registry服务导致端口冲突

## Resolution
- systemctl stop registry && systemctl disable registry
