# harbor密码更改后还被漏扫扫出旧密码

harbor密码更改后漏扫仍报告旧密码

## Cause
- 集群内对应的secret凭据未同步更新密码

## Resolution
- 更新所有使用harbor的secret凭据
- 执行脚本遍历集群secret：kubectl get secret -A >/tmp/1
- cat /tmp/1|awk '{print "bash 1.sh",$1,$2}'|bash >/tmp/result.txt
- 在/tmp/result.txt中搜索旧密码并更新对应secret

## [workaround]
