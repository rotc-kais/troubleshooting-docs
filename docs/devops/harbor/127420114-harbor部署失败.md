# harbor部署失败

harbor部署状态一直显示'部署中' harbor pod处于pending或error状态 PVC创建失败且无持久卷生成

## Cause
- cephfs-provisioner因存储池残留导致重建失败
- cephfilesystem状态非ready
- 存储池存在冲突

## Resolution
- 清理残留存储池
- 重建存储池
- 恢复cephfs-provisioner服务
