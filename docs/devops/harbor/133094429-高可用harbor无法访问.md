# 高可用harbor无法访问

访问harbor无法连接 core日志显示连接pg超时 pg两个节点的日志显示都认为对方是leader

## Cause
- pg副本间出现脑裂（split-brain），导致双主状态

## Resolution
- 手动重启pg集群
