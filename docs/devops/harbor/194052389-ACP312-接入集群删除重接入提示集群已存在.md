# ACP3.12 接入集群删除重接入提示集群已存在

集群重新接入提示集群已存在

## Cause
- 名为tke的cm未清理导致平台判断该集群仍存在

## Resolution
- 通过平台删除对应集群后清理相关资源（cm、svc、ep）
- 清理cpaas-system等平台相关ns中的残留资源
- 删除名为tke的configmap后重新接入集群
