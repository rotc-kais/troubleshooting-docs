# 3.8.1

部署topolvm报错找不到镜像 镜像拉取版本错误

## Cause
- UI清理操作同时删除了后端存储配置
- 重新部署时使用了旧集群配置中的错误镜像版本

## Resolution
- 手动修改yaml镜像版本为csi-snapshotter:v3.0.2和topolvm:3.8.3
- 卸载topolvm-operator并删除其subscription
