# harbor镜像仓库无法进行垃圾清理

harbor镜像仓库无法进行垃圾清理 执行垃圾清理删除镜像报错500 jobservice日志存在特定报错

## Cause
- Harbor自动垃圾收集机制异常

## Resolution
- 手动在registry服务pod执行命令: registry garbage-collect --delete-untagged=true /etc/registry/config.yml
