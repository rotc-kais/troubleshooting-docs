# 持续发布阶段库无数据

持续发布阶段库无数据 无法创建持续发布

## Cause
- 未安装katanomi operator和tekton operator
- 未部署katanomis和tektonpipeline实例

## Resolution
- 安装katanomi operator与tekton operator
- 部署katanomis和tektonpipeline实例
