# 持续发布扫描镜像漏洞失败

执行镜像扫描时报错，http镜像仓库被识别为https

## Cause
- 流水线错误将http协议镜像仓库识别为https协议

## Resolution
- 在镜像仓库配置中显式添加http://协议前缀
