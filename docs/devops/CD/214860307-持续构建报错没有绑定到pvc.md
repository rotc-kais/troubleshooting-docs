# 持续构建报错没有绑定到pvc

pod has unbound immediate persistentVolumeClaims

## Cause
- 缺少默认存储类，导致PersistentVolumeClaims无法自动绑定

## Resolution
- 在平台管理 存储类管理中添加默认存储类

## [workaround]
