# 使用cce集群持续构建找不到pom文件

持续构建报找不到pom文件 容器内无clone的代码信息 容器内无nfs挂载信息

## Cause
- csi-driver-nfs默认kubeletDir配置(/var/lib/kubelet)与CCE实际工作目录(/mnt/paas/kubernetes/kubelet)不一致

## Resolution
- 修改csi-driver-nfs的kubeletDir配置为/mnt/paas/kubernetes/kubelet
