# 持续构建触发器不生效

持续构建配置触发器后没有生效 git仓库中webhook test push返回404状态

## Cause
- 业务集群部署katanomi时ExternalURL填写有误

## Resolution
- 检查并修正katanomi的ExternalURL配置，确保填写katanomi-plugin的地址（支持Ingress或NodePort方式）
