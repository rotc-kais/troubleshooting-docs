# 持续构建执行异常

持续构建push阶段无报错提示终止 任务pod因OOM内存不足被终止

## Cause
- Dockerfile新增较大程序包导致内存需求增加
- 流水线push阶段内存配置不足

## Resolution
- 更新流水线push阶段内存默认值并适当增加配置
