# devops流水线点击执行没有反应

ui点击执行没有反应接口无报错 复制流水线创建会报错500 devops-api无报错日志开启debug发现请求都是200

## Cause
- global节点单点部署
- global节点根磁盘空间占满导致容器异常

## Resolution
- 清理global节点磁盘空间
- 重启异常容器
