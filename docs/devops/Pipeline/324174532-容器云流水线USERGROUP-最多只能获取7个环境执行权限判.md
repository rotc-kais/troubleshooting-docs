# 容器云流水线USERGROUP 最多只能获取7个，环境执行权限判

用户已加入多个用户组但流水线日志显示仅7个用户组 环境执行权限判断未通过

## Cause
- USER_GROUP取值逻辑仅获取关键词后8条数据
- 实际用户组数据分布在关键词后200多行位置

## Resolution
- 调整用户组数据获取范围为50-100行
