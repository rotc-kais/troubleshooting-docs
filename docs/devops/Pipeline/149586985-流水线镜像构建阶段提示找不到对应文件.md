# 流水线镜像构建阶段提示找不到对应文件

流水线镜像构建阶段提示找不到对应文件

## Cause
- 客户自定义的dockerfile内未定义jar包环境变量
- 环境变量设置内容为空

## Resolution
- 修改dockerfile设置对应环境变量为正确的程序包名称
