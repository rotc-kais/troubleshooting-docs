# 流水线使用nexus执行失败

流水线使用nexus执行报错下载依赖失败 内网nexus下载依赖使用releases库 外网直接拉取走public库

## Cause
- 项目依赖包的pom文件指定了固定的外部地址去拉取

## Resolution
- 修改pom文件中的字段去掉nexus地址
- 通过settings.xml管理依赖
