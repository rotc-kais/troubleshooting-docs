# 流水线模版同步镜像修改tag不生效

修改镜像tag后目标镜像标签不生效

## Cause
- 流水线同步镜像模版标签采用1:1机制无法修改tag

## Resolution
- 升级到3.8版本使用新版同步镜像模版

## [workaround]
- 使用图形化流水线创建同步镜像任务修改标签
