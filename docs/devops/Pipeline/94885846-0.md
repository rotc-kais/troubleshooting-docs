# 0

无法找到图形化创建流水线操作菜单

## Cause
- 图形化创建流水线对应的alpha功能开关未启用

## Resolution
- 访问<平台访问地址>/console-platform/home/feature-gate打开平台成熟度设置
- 启用图形化创建流水线对应的alpha功能开关
- 返回DevOps产品界面验证菜单显示状态

## [workaround]

## [Related Information]
- Environment: 3.0以上版本
- /console-platform/home/feature-gate
- 平台成熟度设置
- alpha功能开关
