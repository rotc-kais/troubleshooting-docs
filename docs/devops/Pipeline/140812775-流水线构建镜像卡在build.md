# 流水线构建镜像卡在build

流水线构建镜像卡在build阶段 push镜像报错500 harbor-database容器重启报错no space left on disk

## Cause
- harbor pvc使用的nfs存储磁盘空间使用100%

## Resolution
- 扩容harbor使用的磁盘
