# 3.9

jenkins 流水线卡在clone 这一步，无法进行下一步 手动在jenkins slave节点tools容器或宿主机执行git clone成功 修改后的jenkinsfile无日志输出，流水线未执行

## Cause
- jenkins随机数导致clone步骤偶发性卡住

## Resolution
- 升级到3.10版本修复该问题
