# 3.6.1

DevOps流水线构建失败 java pod调度主机磁盘使用率达到80%触发驱逐

## Cause
- 主机/var/lib/docker目录磁盘使用率过高

## Resolution
- 清理退出容器等资源
- 扩容主机/var/lib/docker目录并单独挂盘（建议不少于100G）

## [workaround]
- 手动清理容器资源临时恢复流水线
