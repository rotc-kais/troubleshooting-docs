# 流水线编译失败

流水线编译日志报错webpack文件权限不足 依赖加载异常

## Cause
- node_modules/.bin目录权限不足
- nodejs版本不兼容（客户使用v12 vs 平台默认v22）

## Resolution
- 在Dockerfile添加RUN chmod +x node_modules/.bin/*
- 使用nodejs 12版本节点执行流水线
