# 执行流水线无法选择分支

执行流水线无法选择分支

## Cause
- 旧gitlab迁移后凭据权限异常导致集成资源异常

## Resolution
- 在新gitlab中集成repo
- 流水线配置中切换至新gitlab集成资源
- 删除旧gitlab集成资源
