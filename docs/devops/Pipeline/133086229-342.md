# 3.4.2

更新流水线页面报错: the server is currently unable to handle the request (post pipelines.devops.alauda.io)

## Cause
- devops-apiserver组件处理devops组API资源查询时间过长
- 日志中存在异常抛出

## Resolution
- 重启devops-apiserver组件
- 检查并优化API资源查询性能

## [workaround]
- 立即重启devops-apiserver组件
