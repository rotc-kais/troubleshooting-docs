# 流水想运行报错

流水线执行失败 kubectl 命令请求超时

## Cause
- 节点挂掉导致 apiserver 状态异常
- 请求负载到异常节点的 apiserver

## Resolution
- 启动挂掉的节点
- 确保节点状态变为 Ready
