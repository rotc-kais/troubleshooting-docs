# buildah构建缓慢

buildah构建镜像时copy包过程缓慢，导致流水线执行时间长或失败

## Cause
- 默认使用fuse-overlayfs文件系统导致copy操作性能问题

## Resolution
- 构建参数添加--storage-driver=vfs
- 推送参数添加--storage-driver=vfs
