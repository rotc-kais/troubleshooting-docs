# 3.12

流水线build报错拉取依赖失败 容器内curl依赖地址不通 不同节点/etc/resolv.conf中nameserver不一致

## Cause
- coredns所在节点无法联通依赖地址
- 节点间DNS配置(nameserver)不一致

## Resolution
- 修改其他节点的/etc/resolv.conf与nodejs节点配置一致
- 重启coredns服务
