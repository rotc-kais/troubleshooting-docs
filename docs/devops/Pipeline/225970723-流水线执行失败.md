# 流水线执行失败

流水线执行失败-报错上传nexis无secret导致上传失败

## Cause
- nexus文件夹名称大写导致

## Resolution
- 重新创建小写名称的nexus文件夹，在平台侧进行授权使用，重新执行流水线
