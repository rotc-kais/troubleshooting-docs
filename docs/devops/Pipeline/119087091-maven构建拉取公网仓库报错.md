# maven构建拉取公网仓库报错

maven构建拉取公网仓库报错 容器内curl能解析但超时 流水线运行报超时错误

## Cause
- 未指定maven仓库地址
- 容器网络无法通过代理访问外网

## Resolution
- 在直连外网的机器部署nexus仓库代理公网依赖
- 配置服务器通过网卡直连外网
