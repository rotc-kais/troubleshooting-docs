# 实验环境里配置了流水线的源，但是在流水线任务中没有生效

流水线源配置未生效 特定task（发送微信）未显示或执行失败

## Cause
- task的apiVersion配置错误（非v1beta1）

## Resolution
- 修正yaml中task的apiVersion为v1beta1
