# 脚本流水线参数化构建弹窗不显示

脚本流水线参数化构建弹窗不显示 基于老流水线复制或新建流水线不显示参数化构建 新建简单流水线可正常显示参数化构建

## Cause
- 流水线包含复杂函数或内置option配置
- ACP控制器同步机制影响参数化构建显示

## Resolution
- 1. 新建简单流水线模板
2. 将旧流水线内容迁移至新模板
