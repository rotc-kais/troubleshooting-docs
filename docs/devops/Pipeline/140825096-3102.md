# 3.10.2

CICD打包python镜像报错gcc命令下载失败 使用阿里云镜像源下载速度较慢

## Cause
- dockerfile内使用的阿里云镜像源下载速度慢导致gcc安装失败

## Resolution
- 修改dockerfile内基础镜像的镜像源为清华大学/网易等国内镜像源
