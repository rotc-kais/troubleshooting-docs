# 对接外部nexus不显示项目

对接外部nexus不显示项目

## Cause
- nexus.properties中未配置nexus.scripts.allowCreation=true

## Resolution
- 在nexus.properties中添加/修改配置项nexus.scripts.allowCreation=true
