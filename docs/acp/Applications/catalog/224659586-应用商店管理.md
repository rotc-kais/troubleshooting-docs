# 应用商店管理

应用商店无法识别arm64架构的应用包 部署时无法选择该模板或模板版本

## Cause
- 应用商店通过Chart.yaml的annotations字段识别架构支持

## Resolution
- 修改Chart.yaml添加annotations: cpaas.io/arch: amd64.arm64
- 执行helm package重新打包应用
- 删除旧应用包并上传新应用包
