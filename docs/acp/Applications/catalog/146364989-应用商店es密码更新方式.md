# 应用商店es密码更新方式

401提示 集群状态不健康

## Cause
- ES密码变更未生效
- 未正确更新加密字典

## Resolution
- 检查命名空间加密字典获取当前ES认证信息
- 执行curl -u 用户名:密码 'http://localhost:9200/_cluster/health?pretty'验证服务状态
- 通过平台更新ES加密字典内容
- 在应用界面重启ES服务并等待Pod就绪
- 使用curl -u 用户名:新密码 'http://localhost:9200/_cluster/health?pretty'验证新密码
