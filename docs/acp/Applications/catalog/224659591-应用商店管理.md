## Cause

## Resolution
- 在Chart.yaml的annotations中添加架构支持：
annotations:
  cpaas.io/arch: amd64.arm64
- 在Chart.yaml的annotations中添加网络栈支持：
annotations:
  cpaas.io/protocol-stack: IPv4
