# 应用分发导入后创建新应用环境变量值发生改变

环境变量的值在创建新应用时发生变化

## Cause
- TKE 3.8版本未对chart values中的值进行转义处理
- helm未将未转义的值识别为json字符串

## Resolution
- 升级到TKE 3.10版本(已修复该问题)

## [workaround]
- 在创建应用时手动设置经过正确转义的values值
