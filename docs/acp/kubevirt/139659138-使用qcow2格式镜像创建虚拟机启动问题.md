# 使用qcow2格式镜像创建虚拟机启动问题

使用qcow2格式镜像创建的虚拟机无法进入系统 虚拟机能够正常启动但无法登录

## Cause
- 镜像不支持virtio磁盘驱动
- 平台默认使用virtio磁盘总线类型

## Resolution
- 修改虚拟机YAML配置中的磁盘总线类型：
    domain:
      devices:
        disks:
        - disk:
            bus: sata
          name: rootdisk

## [workaround]
