# 3.10.1

虚拟机重启后一直处于'准备中'状态 对应PVC处于Pending状态

## Cause
- 新增700Gi数据盘超过topolvm存储容量

## Resolution
- kubectl get dvs -A 查找对应datavolume并执行delete操作
- kubectl edit vms <vm name> 删除disk相关YAML片段(需修改两处)
- 通过TKE界面停止并重新启动虚拟机
