# 虚拟机报错文件系统损坏无法进入

虚拟机报错文件系统损坏无法进入 挂载测试验证文件系统时确认损坏

## Cause

## Resolution
- 执行 xfs_repair /dev/rdb1p1 进行修复
- 执行 xfs_repair -L /dev/rdb1p1 强制修复（需提前备份）
