# 3.6

部分pod处于creating状态 cni报错ping网关not ready master节点上的pod一直处于creating状态

## Cause
- 虚拟机混杂模式未开启

## Resolution
- 参考OPS-94887239-091021-1526-1243.pdf文档开启虚拟机混杂模式
