# 虚拟机热迁移咨询问题

无法执行虚拟机热迁移操作 节点故障时缺乏自动迁移机制

## Cause
- ACP产品设计未包含热迁移功能
- 与OpenStack架构设计理念不同

## Resolution

## [workaround]
- 采用传统高可用方案(如备份恢复)
- 通过手动迁移虚拟机实例
- 考虑使用支持热迁移的虚拟化平台
