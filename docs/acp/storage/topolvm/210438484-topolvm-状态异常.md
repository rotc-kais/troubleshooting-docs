# topolvm 状态异常

topolvm-prepare-vg组件日志显示vdc磁盘不是裸盘状态 存储节点状态异常 无法显示容量

## Cause
- vdc磁盘未处于裸盘状态导致无法创建vg

## Resolution
- 更换磁盘并重新部署topolvm

## [workaround]
- 将存储节点磁盘恢复为裸盘状态后重新部署
