# 3.10.1

topolvm-node pod处于pending状态 节点存在磁盘压力污点 /盘inode使用超过95%

## Cause
- /盘inode使用率超过95%触发磁盘压力标签
- 测试部署的minio插件导致/data目录inode异常

## Resolution
- 执行命令检查inode使用情况: for i in /*; do echo $i; find $i | wc -l; done
- 卸载minio插件
- 清理/data目录异常文件
