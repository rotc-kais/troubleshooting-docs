# 3.8

部署minio插件失败 minio的pod一直处于Terminating状态

## Cause
- 历史部署残留资源未清理
- 磁盘路径'/export-1'与系统根目录相同

## Resolution
- 前端卸载minio插件
- 清理minio对应的job和sts资源
- 确认资源清理完成后重新部署
