# rook

运维中心页面中无法看到与Ceph相关的告警策略和告警信息

## Cause
- v3.14.2版本中rook-ceph/topolvm/minio存储的告警策略在运维中心告警策略页面不显示是产品设计现状

## Resolution
- 升级到4.0.3及以上版本(运维中心已内置告警规则)
- 历史版本需手动应用告警规则(需替换CLUSTER和NAMESPACE参数)

## [workaround]
- 手动创建告警规则并替换CLUSTER和NAMESPACE参数
