# 3.12.1

通过minio UI界面下载资源失败 minio-console日志存在报错 匿名下载未成功

## Cause
- 目标bucket未配置匿名下载权限
- minio-console存在已知下载功能缺陷

## Resolution
- 执行存储桶匿名下载授权命令：mc anonymous set download buckup/xxxx
