# ceph

mon is allowing insecure global_id reclaim

## Cause
- auth_allow_insecure_global_id_reclaim enabled

## Resolution
- kubectl edit csv -n rook-ceph rook-ceph.v3.8.4
- ceph config set mon auth_allow_insecure_global_id_reclaim false
