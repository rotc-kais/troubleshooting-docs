# 3.10.2

ceph集群健康状态告警 mon节点df -hT查询夯住 mon组件存在多余pending节点

## Cause
- rook-ceph-mon-endpoints和rook-ceph-csi-config configmap存在重复节点配置

## Resolution
- 修改rook-ceph-mon-endpoints和rook-ceph-csi-config configmap删除多余节点
- 删除对应的mon deployment
- 重启异常ceph组件
