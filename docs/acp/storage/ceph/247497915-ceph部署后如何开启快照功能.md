# ceph部署后如何开启快照功能

部署后无法通过界面进行快照的开启

## Cause
- 默认创建Ceph集群时未启用快照功能

## Resolution
- 通过集群-插件-部署卷快照插件
- 创建VolumeSnapshotClass资源并应用指定YAML配置
- 更新CephCluster资源增加cpaas.io/ceph-snapshot-enabled: "true"注解
