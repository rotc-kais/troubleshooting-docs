# 创建存储统声明无对象存储类可进行选择

创建存储统声明无对象存储类可进行选择

## Cause
- 未对接对象存储类
- 未部署容器对象存储接口插件-COSI
- 未部署Ceph对象存储卷插件

## Resolution
- 完成对象存储对接
- 在集群插件中心部署COSI插件
- 部署Ceph对象存储卷插件
