# 数据服务使用rook

备份策略执行失败 rook-ceph对象存储pod出现OOM

## Cause
- 对象存储pod默认资源配置不足（CPU 500M，内存1Gi）导致OOM

## Resolution
- 更新CephObjectStore的资源配置，增加CPU和内存限制

## [workaround]
