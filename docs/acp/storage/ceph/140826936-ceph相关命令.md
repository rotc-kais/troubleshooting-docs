# ceph相关命令

集群状态异常 OSD/MON/MDS组件故障 存储池配置问题

## Cause
- OSD被逐出或down状态
- MON节点异常
- PG数量不足或分布不均
- 存储池副本数/空间限制配置错误

## Resolution
- ceph osd in osd.3
- ceph osd setmaxosd 10
- ceph osd pool set data pg_num 100
- ceph mon repair
- ceph osd crush remove osd.4

## [workaround]
- ceph osd pause
- ceph osd unpause
- ceph osd pool set data min_size 2
