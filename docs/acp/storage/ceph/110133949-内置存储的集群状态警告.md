# 内置存储的集群状态（警告）

集群状态显示为HEALTH_WARN mons b,c are low on available space /var/lib磁盘空间使用率超过80%

## Cause
- /var/lib/ceph/mon路径所在磁盘可用空间低于30%

## Resolution
- 扩容/var/lib目录所在磁盘空间
