# 国泰君安在使用网络分离的方法部署 ceph 后, 只能在集中式网关的配置下部署 pvc. 分布式网关部署会失败

集中式网关部署时 ceph 无法使用，某两个组件之间的网络不通 分布式网关部署正常

## Cause
- 网关出口 ip 配置是实验性功能，未经完整测试

## Resolution
- 在自有环境尝试重现问题，从 ceph 的二层地址和网关配置路径开始排查
