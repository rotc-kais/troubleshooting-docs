# 3.8.2

rgw组件pod健康检查失败不断重启 退出码137 日志无显著报错仅有debug信息

## Cause
- 部署时存储设备选择配置添加了多个磁盘

## Resolution
- 每个节点添加单个存储设备后重新部署
