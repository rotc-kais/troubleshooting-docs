# rook

rook-ceph告警monitor存在的ceph版本个数大于1 mon.d实例缺失ceph_version字段 mon-d容器存在频繁重启记录

## Cause
- mon-d容器异常导致mgr无法获取监控数据
- 缺失ceph_version字段的monitor被统计为独立版本

## Resolution
- 重启mon-d和mgr容器恢复监控数据采集
