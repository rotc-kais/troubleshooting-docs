# ceph状态告警

ceph界面状态告警 执行ceph -s/ceph health detail报错mon crashed on host

## Cause
- mon容器异常崩溃

## Resolution
- 在rook-ceph-tools容器内执行ceph crash ls
- 使用ceph crash rm <id>删除对应崩溃记录
