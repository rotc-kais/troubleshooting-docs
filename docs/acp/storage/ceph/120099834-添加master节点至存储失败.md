# 添加master节点至存储失败

无法获取节点上可用设备，显示无可用存储设备

## Cause
- master节点缺少device-discover或rook-discover组件
- master节点存在noscheduler污点

## Resolution
- 删除master节点的noscheduler污点
- 重启topolvm-operator
- 将discover组件部署到master节点

## [workaround]
