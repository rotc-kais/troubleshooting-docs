# 3.4.2

rook-ceph创建后告警 无法创建PV

## Cause
- OSD配置中磁盘名称错误地包含/dev前缀

## Resolution
- 修改cephcluster配置，将磁盘名称改为具体磁盘名称(如sdb)而非/dev/sdb格式
