# 内置存储

clock skew detected on mon.b, mon.c

## Cause
- mon节点上ntp服务器未启动
- ceph设置的mon时间偏差阈值较小(默认0.05s)

## Resolution
- 对所有节点进行ntp服务部署，指定统一的时间同步服务器
