# ceph故障

ceph状态显示故障 三节点osd全部down掉 磁盘盘符从sdc变为sda

## Cause
- 物理机重启导致磁盘盘符发生改变

## Resolution
- 调整rook-ceph-operator和rook-ceph-osd副本数为0
- 备份并删除原有PV
- 修改PV配置中的设备路径(将/dev/sdc改为/dev/sda)
- 重建PV并修复PVC绑定状态
- 恢复operator和osd副本数
