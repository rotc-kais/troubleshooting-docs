# 集群插件看不到nfs插件安装选项

集群插件看不到nfs插件安装选项

## Cause
- 3.12版本只检查nfs csi插件且下线了provisioner
- 升级时未执行nfs-provisioner到nfs-csi的迁移

## Resolution
- 执行nfs-provisioner到nfs-csi的迁移
- 参考迁移指南: https://confluence.alauda.cn/pages/viewpage.action?pageId=123602413
