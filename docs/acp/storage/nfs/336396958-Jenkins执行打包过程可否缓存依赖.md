# Jenkins执行打包过程可否缓存依赖

流水线执行时卡在‘等待调度任务’阶段 配置Maven构建缓存后流水线无法正常执行

## Cause
- topolvm存储类与缓存配置存在兼容性问题
- 未正确配置持久化存储

## Resolution
- 参考DevOps流水线文档配置缓存路径
- 检查PVC配置与存储类兼容性
- 调整持久卷声明(PVC)设置

## [workaround]
- 暂时禁用依赖缓存功能
- 改用兼容的存储类替代topolvm
