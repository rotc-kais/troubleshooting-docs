# 3.12.1

节点上发现大量的僵尸进程，父进程为nfs存储类插件

## Cause
- nfs存储类插件在mount操作异常时产生rpcd僵尸进程
- NFS客户端与服务器之间的网络通信中断或超时

## Resolution
- 在存储类添加mount option '-o nolock'
- 增加NFS挂载参数timeo=300和retrans=5
