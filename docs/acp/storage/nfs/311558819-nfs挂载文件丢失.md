# nfs挂载文件丢失

PVC2挂载至`/test/test1`目录发生解挂导致目录丢失 父目录`/test`保持挂载状态

## Cause
- NFS服务端断链
- 挂载参数配置不当(timeo=600/retrans=2)
- 子目录挂载方式触发NFS机制

## Resolution
- 调整StorageClass挂载参数：timeo=1200/retrans=3
- 在容器中运行后台进程保持对关键子目录的占用
- 优化NFS服务端稳定性配置
