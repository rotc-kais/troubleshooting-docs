# nfs存储创建PVC超时失败

global环境创建PVC失败，region环境成功 csi-nfs-controller组件日志中存在连接超时报错 主机上手动挂载NFS正常，但PVC无法绑定

## Cause
- global集群rpc-statd服务未正常运行

## Resolution
- 启动global集群所有节点的rpc-statd服务
