# 对接商业nas存储后创建PVC失败

创建PVC失败，报错"failed to provision volume with storageclass xxx : rpc error: context deadline exceeded" csi-nfs-node容器日志报错"rpc.statd is not running"

## Cause
- 节点rpcbind服务未启动

## Resolution
- 在节点上启动rpcbind服务
