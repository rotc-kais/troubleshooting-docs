# 3.6

nfs-updater组件运行异常 kubectl get minfo -A显示创建人邮箱与用户邮箱不一致

## Cause
- 创建nfs的账户邮箱被更改导致原用户不存在

## Resolution
- 手动编辑集群nfs的hr资源，将创建者改为平台admin
