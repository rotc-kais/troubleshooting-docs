# 20220610

kube-ovn-cni pod状态异常 节点因压力大被标记为不可调度 pod describe显示readiness和liveness检查失败

## Cause
- 磁盘I/O性能问题导致apiserver访问超时

## Resolution
- 修改kube-ovn-cni DaemonSet配置，删除livenessProbe检查
