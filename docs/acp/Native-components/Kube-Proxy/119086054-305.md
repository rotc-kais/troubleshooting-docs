# 3.0.5

alb 80端口不通 主机上telnet本机80端口不通 telnet localhost 80正常但127.0.0.1和本机IP 80不通

## Cause
- 主机存在其他应用占用80端口

## Resolution
- 删除占用主机80端口的业务应用
- 重启kube-proxy组件
