# 业务使用SVC无法访问

curl svc:port 超时 curl ep:port 超时 ping pod ip 能通

## Cause
- 节点 firewalld 防火墙拦截了服务通信

## Resolution
- 关闭防火墙: systemctl stop firewalld
- 禁止防火墙开机自启: systemctl disable firewalld
