# uos系统

部署集群卡在k8s阶段 报错admin.conf文件问题 kubeadm init阶段6443端口TLS握手失败

## Cause
- UOS系统内核参数缺失导致sysctl -p加载失败
- /etc/sysctl.d/99-tke.conf中的内核参数未生效

## Resolution
- 手动执行sysctl -p /etc/sysctl.d/99-tke.conf -e加载内核参数
