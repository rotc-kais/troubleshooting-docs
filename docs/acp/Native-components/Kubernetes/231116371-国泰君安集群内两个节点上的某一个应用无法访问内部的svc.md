# 国泰君安，集群内两个节点上的某一个应用无法访问内部的svc

跨节点访问svc不通 仅特定节点上的client pod访问特定server pod时异常 报文在server节点underlay网卡未捕获

## Cause
- underlay网络层报文转发异常

## Resolution
- 联系iaas团队排查网络报文转发问题

## [workaround]
- 创建测试pod/svc延长故障排查时间窗口
