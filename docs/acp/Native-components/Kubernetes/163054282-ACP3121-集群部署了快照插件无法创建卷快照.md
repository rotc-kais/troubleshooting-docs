# ACP3.12.1 集群部署了快照插件，无法创建卷快照

PVC卷快照处显示未部署插件

## Cause
- 缺少csi-topolvm-snapshotclass VolumeSnapshotClass资源

## Resolution
- 创建csi-topolvm-snapshotclass VolumeSnapshotClass资源：kubectl create -f <yaml文件>

## [workaround]
- 通过存储管理界面启用快照功能
