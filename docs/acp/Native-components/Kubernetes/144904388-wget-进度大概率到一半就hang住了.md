# wget 进度大概率到一半就hang住了

wget下载进度到一半停止 无TCP重传或丢包 内核报错TCP: out of memory

## Cause
- net.ipv4.tcp_mem配置过低导致内存耗尽

## Resolution
- sysctl -w net.ipv4.tcp_mem="196608 262144 393216"
