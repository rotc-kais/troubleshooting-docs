# 容器能不能配置自动重启

JIRA Jira：

## Cause

## Resolution
- 使用 Kubernetes CronJob 创建定时任务删除并重建 Pod
- 配置 resources.limits.memory 和 livenessProbe 检查内存使用情况
- 结合 Horizontal Pod Autoscaler 实现自动扩展
