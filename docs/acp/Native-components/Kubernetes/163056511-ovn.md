# ovn

ovn-central pod启动失败 192.168.32.1:443 i/o timeout错误 节点curl 192.168.32.1:443概率性不通(1/5)

## Cause
- 客户护网期间添加的iptables规则导致6443端口流量被DROP

## Resolution
- 清理护网期间添加的iptables规则
