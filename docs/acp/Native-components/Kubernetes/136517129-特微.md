# 特微

节点200.13上的pod无法访问其他节点pod 12/14节点pod访问正常

## Cause
- underlay配置使用了eno12399网卡
- IaaS层对eno12399网卡存在跨节点通信限制

## Resolution
- 临时将overlay通信网卡改为管理网卡eno0800
- 待IaaS层修复eno12399网卡问题后恢复配置
