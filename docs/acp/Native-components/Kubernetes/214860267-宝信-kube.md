# 宝信 kube

kube-ovn-cni pod 不断 crash 业务 pod 网络受影响 IPv6 路由异常删除/添加失败

## Cause
- IPv6 访问不通导致路由异常
- 代码逻辑缺陷导致路由误删

## Resolution
- 手动添加 IPv6 路由
- 重新配置 ovn0 网卡的 IPv6 地址触发自动路由生成

## [workaround]
- 重启 master 节点和部分 worker 节点
