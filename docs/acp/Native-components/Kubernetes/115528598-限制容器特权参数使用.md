# 限制容器特权参数使用

JIRA Jira：

## Cause
- 平台默认允许容器特权参数(privileged: true)的使用

## Resolution
- 3.8之前版本: 容器平台管理视图-安全管理-容器组安全策略-立即配置勾选限制
- 3.8及之后版本: 平台管理视图-安全管理-容器组安全策略-立即配置勾选限制
