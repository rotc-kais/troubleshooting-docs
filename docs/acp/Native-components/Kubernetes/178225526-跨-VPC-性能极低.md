# 跨 VPC 性能极低

跨 VPC 通过 Pod IP 调用 HTTP 接口下载 5MB 文件耗时 5~10s 同 VPC 内调用不到 1s

## Cause
- VXLAN 封装包出现乱序导致频繁丢包及重传

## Resolution
- 由 Iaas 排查 VXLAN 乱序问题
