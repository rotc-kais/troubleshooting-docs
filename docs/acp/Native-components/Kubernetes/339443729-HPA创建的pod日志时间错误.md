# HPA创建的pod日志时间错误

HPA创建的pod日志时间显示为两天 ES索引保存时间与设置生命周期不符

## Cause
- Kubernetes生成的UTC时间与应用本地时间(北京时间)存在时区差异

## Resolution
- 统一应用日志时区与Kubernetes时区配置
- 检查ES索引生命周期策略与日志时间格式的对应关系

## [workaround]
- 在应用侧增加时区标识或转换时间格式
- 调整ES生命周期策略时间窗口

- Environment: 3.12.2
