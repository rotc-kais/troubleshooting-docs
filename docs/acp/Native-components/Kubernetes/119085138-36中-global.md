# 3.6中 global

收到global-asm-serving-cert平台的证书告警信息，提示还有30天到期

## Cause
- 证书默认签发周期为90天，提前30天触发续签告警

## Resolution
- 确认该证书由cert-manager自动管理更新
- 通过命令 kubectl get certificate global-asm-serving-cert -n cpaas-system -oyaml 查看证书到期时间、上次更新时间和下次更新时间
