# 节点宕机后状态变为notready 时间过长40

节点宕机后状态变为notready时间过长（40-100s）

## Cause
- kube-controller-manager leader节点持有lease对象期间发生故障，导致lease续约失败
- 新leader选举需等待lease持续时间的一半（默认7.5秒）
- 新leader选举后node-monitor-grace-period参数需重新开始计算时间
