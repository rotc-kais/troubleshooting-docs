# 容器内低熵情况

java应用生成随机数时阻塞

## Cause
- 容器内系统熵池不足导致/dev/random阻塞

## Resolution
- 安装rng-tools或haveged增加系统熵

## [workaround]
- 改用/dev/urandom替代/dev/random
