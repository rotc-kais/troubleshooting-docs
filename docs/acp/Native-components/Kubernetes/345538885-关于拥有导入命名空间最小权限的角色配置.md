# 关于拥有导入命名空间最小权限的角色配置

JIRA Jira：

## Cause

## Resolution
- 使用PATCH方法调用API：/kubernetes/${CLUSTER_NAME}/api/v1/namespaces/${NAMESPACE}，携带{"metadata":{"labels":{"cpaas.io/project":"${PROJECT_NAME}"}}}请求体
- 通过平台管理员复制角色进行改造，添加命名空间patch权限及相关自定义权限(具体配置需参考截图)
