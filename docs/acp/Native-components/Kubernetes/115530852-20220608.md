# 20220608

ovs-ovn pod启动失败 日志显示无法连接ovn sb db kubectl sbctl show缺少问题节点的chassis信息

## Cause
- 残留的旧DB文件导致连接异常

## Resolution
- 删除/etc/origin/openvswitch/conf.db文件并重启pod
