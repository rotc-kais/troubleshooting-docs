# acp 3.6.2 ovn

k8s svc lb 结果未准确轮询调度 ovn lb 结果不如预期均衡

## Cause
- ovn lb 基于 hash 算法实现

## Resolution
- 关闭 ovn lb 改用 ipvs 进行负载均衡
