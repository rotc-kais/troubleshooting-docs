# 服务网关

网关状态一直处于处理中 Service "gateway" not found

## Cause
- NodePort主机端口配置为38080，超过k8s默认NodePort范围(30000-32767)

## Resolution
- 重新部署网关并将NodePort主机端口配置为30080
