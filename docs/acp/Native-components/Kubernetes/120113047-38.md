# 3.8

日志存储组件资源不足

## Cause
- cpaas-elasticsearch deployment资源配置不足

## Resolution
- 更新cpaas-elasticsearch deployment的resources配置
- 调整ES_JAVA_OPTS环境变量：-Xms8g -Xmx8g（需根据实际memory大小设置）
