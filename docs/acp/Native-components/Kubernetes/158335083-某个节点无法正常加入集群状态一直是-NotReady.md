# 某个节点无法正常加入集群，状态一直是 NotReady

节点状态为 NotReady kube-ovn-cni 日志显示节点与 join 网关无法正常通信 ovn-controller 日志中存在无法连接 sb 的报错

## Cause
- 外部网络未开放故障节点与 SB 之间的 TCP 端口

## Resolution
- 调整外部网络策略，开放故障节点与 SB 之间的 TCP 端口
