# TKE创建新集群失败

页面报错 error trying to reach service: stream error: stream ID 13; INTERNAL_ERROR; received from peer 添加节点失败

## Cause
- api-server 的证书过期需要更新

## Resolution
- 更新 api-server 证书后重启其 pod 同步信息
