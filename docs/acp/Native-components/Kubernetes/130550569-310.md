# 3.10

日志采集失败 es及kafka无数据写入且无报错

## Cause
- 业务集群缺少callback-secret和log-center-secret资源
- cluster-transformer未同步secret资源
- 3.10.0版本与k8s 1.24存在兼容性问题

## Resolution
- 升级平台版本到3.10.1

## [workaround]
- 给razor添加参数--legacy-callback：kubectl edit deploy -ncpaas-system razor
