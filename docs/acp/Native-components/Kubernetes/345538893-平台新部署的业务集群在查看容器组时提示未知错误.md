# 平台新部署的业务集群，在查看容器组时提示未知错误

查看容器组时提示未知错误-117502 Apollo日志显示访问了错误的资源（如meta.k8s.io）

## Cause
- client缓存和schema初始化顺序问题导致访问错误的GVK

## Resolution
- 重启Apollo服务
