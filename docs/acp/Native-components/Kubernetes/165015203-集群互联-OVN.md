# 集群互联 OVN

跨集群 Pod IP 不通

## Cause
- 重建集群后重新加入集群互联导致旧 chassis 资源冲突

## Resolution
- 在 ovn-ic-db 容器(leader)执行 ovn-ic-sbctl destroy gateway/availability_zone xxxx 清理冲突资源
