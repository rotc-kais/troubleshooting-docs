# 3.10.2

Platform subnet management unavailable in UI

## Cause
- Missing cni-kube-ovn ConfigMap in standard k8s clusters with OVN network plugin

## Resolution
- Manually create cni-kube-ovn ConfigMap in kube-public namespace
