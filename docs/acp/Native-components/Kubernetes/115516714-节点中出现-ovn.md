# 节点中出现 ovn

存在 ovn-nbctl defunct 僵尸进程

## Cause
- kube-ovn-controller 未正确处理 SIGCHLD 信号等待子进程退出

## Resolution
- 手动重建 kube-ovn-controller
- 更新 kube-ovn 版本

## [workaround]
- 手动重建 kube-ovn-controller
