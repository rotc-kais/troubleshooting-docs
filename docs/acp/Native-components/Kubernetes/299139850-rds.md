# rds

内存不足的错误提示 内存使用率在重启前达到95%以上 服务因OOM被终止

## Cause
- rds-api服务配置的内存限制不足

## Resolution
- 调整rds-api的Pod资源限制增加内存配额
- 优化服务内存使用减少不必要的缓存
- 设置合理的HPA策略
- 加强监控告警在内存使用率80%预警
