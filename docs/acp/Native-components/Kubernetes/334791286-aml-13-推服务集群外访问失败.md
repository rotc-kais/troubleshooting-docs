# aml 1.3  推服务集群外访问失败

集群外访问失败 1.2 的 Ingress 配置在 1.3 中不存在

## Cause
- AML 1.3 移除了原有的 Ingress 配置
- 未采用新的外部访问方案

## Resolution
- 参考文档 https://docs-dev.alauda.cn/ai/main/model_inference/inference_service/how_to/external_access_inference_service.html 配置新方案
