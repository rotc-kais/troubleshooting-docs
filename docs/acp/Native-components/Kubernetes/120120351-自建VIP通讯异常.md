# 自建VIP通讯异常

拉取镜像失败 平台加载异常

## Cause
- /etc/kubernetes/keepalived/check.sh 文件为非root用户所有

## Resolution
- 在global集群3个master节点执行 chown root:root /etc/kubernetes/keepalived/check.sh
- 重启keepalived容器
