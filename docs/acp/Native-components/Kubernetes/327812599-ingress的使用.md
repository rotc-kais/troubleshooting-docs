# ingress的使用

无法创建ingress资源

## Cause
- ingress资源的api版本(networking.k8s.io/v1beta1)与当前k8s版本(networking.k8s.io/v1)不匹配

## Resolution
- 修改apiVersion为networking.k8s.io/v1
- 调整ingress配置：添加ingressClassName字段
- 修改backend结构为service.name + service.port.number格式

## [workaround]
