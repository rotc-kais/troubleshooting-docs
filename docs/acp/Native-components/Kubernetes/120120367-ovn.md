# ovn

内存持续上涨 超过400M触发告警 /var/log/kube-ovn/kube-ovn-pinger.log文件大于1G

## Cause
- pinger日志文件未轮转导致内存持续增长

## Resolution
- 清空日志文件：echo 1 > /var/log/kube-ovn/kube-ovn-pinger.log
- 升级Kube-OVN至v1.8.9版本(含logrotate修复)

## [workaround]
