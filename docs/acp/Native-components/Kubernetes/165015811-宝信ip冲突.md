# 宝信ip冲突

pod创建失败 kube-ovn-controller日志显示相同IP的lsp已被分配

## Cause
- kube-ovn-controller存在多个leader导致IPAM未同步

## Resolution
- 重启kube-ovn-controller pod触发重新选主并初始化IPAM
