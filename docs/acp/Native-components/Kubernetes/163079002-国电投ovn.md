# 国电投ovn

ovn-central全部起不来 删除/etc/orgin/ovn/后仍无法恢复 kubectl操作及删除pod缓慢

## Cause
- 磁盘读写问题（wa过高）

## Resolution
- 删除ovn-central的livenesss探针

## [workaround]
- 删除ovn-central的livenesss探针
