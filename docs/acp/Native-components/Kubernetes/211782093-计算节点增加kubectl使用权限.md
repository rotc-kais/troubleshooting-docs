# 计算节点增加kubectl使用权限

JIRA Jira：

## Cause
- 计算节点缺少kubeconfig配置

## Resolution
- 在计算节点执行 mkdir -p $HOME/.kube
- 从master节点执行 scp /etc/kubernetes/admin.conf root@x.x.x.x:$HOME/.kube/config
- 验证kubectl命令可用性
