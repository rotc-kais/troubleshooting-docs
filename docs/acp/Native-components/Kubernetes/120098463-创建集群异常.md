# 创建集群异常

前端查询集群节点处于添加中状态

## Cause
- 业务集群的oidc-ca.crt同步的证书存在问题

## Resolution
- 手动更新业务集群的oidc-ca.crt证书

## [workaround]

## [Related Information]

- /etc/kubernetes/pki/oidc-ca.crt
