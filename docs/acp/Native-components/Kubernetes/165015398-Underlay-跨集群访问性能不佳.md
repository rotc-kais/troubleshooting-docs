# Underlay 跨集群访问性能不佳

Underlay 性能测试结果比 Overlay 集群互联差，不符合预期

## Cause

## Resolution
- 为每个 VLAN 添加 VLAN 子接口并配置 Underlay 使用 VLAN 子接口，重新部署 Underlay
