# exec: "ovs

pod 无法正常删除 kube-ovn-cni 报错 ovs-vsctl not found in $PATH

## Cause
- 第三方安全工具 G01 导致 kube-ovn-daemon 进程的 PATH 环境变量异常

## Resolution
- 卸载 G01 安全工具
- 重建 kube-ovn-cni
