# 集群内部svc名称进行链接

偶现链接被重置

## Cause
- conntrack模块在严格模式下丢弃无效TCP包导致连接重置

## Resolution
- echo 1 > /proc/sys/net/ipv4/netfilter/ip_conntrack_tcp_be_liberal
- echo net.netfilter.nf_conntrack_tcp_loose = 1 > /etc/sysctl.conf && sysctl -p
