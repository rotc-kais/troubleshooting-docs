# 3.8.1

日志显示无数据 lanaya报错read kafka message error elasticsearch缺少kubernetes/log-system索引

## Cause
- nevermore向kafka写入超过1MB数据导致异常

## Resolution
- 应用razor组件hotfix镜像 build-harbor.alauda.cn/ait/razor:hotfix-kafka-config.2205201801
