# 苏宁银行 ns 导入问题

ns导入失败 ns的annotation不断刷新 subnet的excludeIPs字段持续更新

## Cause
- 子网持续更新excludeIPs字段导致annotation刷新

## Resolution
- 删除excludeIPs字段后恢复
- 建议更新hotfix镜像后再确认
