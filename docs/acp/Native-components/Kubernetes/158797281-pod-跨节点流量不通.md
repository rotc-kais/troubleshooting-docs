# pod 跨节点流量不通

pod跨节点流量不通 nc -vzu测试目标端口6081显示连通

## Cause
- 安全组未放行目标端口6081

## Resolution
- 修改安全组以放行相关端口

## [workaround]
- 创建允许所有流量的安全组进行测试
