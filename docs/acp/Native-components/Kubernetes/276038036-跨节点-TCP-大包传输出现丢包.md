# 跨节点 TCP 大包传输出现丢包

跨节点传输TCP大包时出现丢包 genev_sys_6081网卡DROP统计增加 迁移到ARM裸金属节点后问题消失

## Cause
- 云厂商提供的x86虚拟机环境存在隧道封装性能问题

## Resolution
- 联系IaaS云厂商协调解决

## [workaround]
- 将应用迁移到ARM裸金属节点
