# 3.8.1

operator pod下报warning事件DisabledCopiedCSVs

## Cause
- 升级到3.8.1版本后关闭了csv copy开关

## Resolution
- 忽略该警告事件（事件1小时由Kubernetes自动回收）
- 将disableCopiedCSVs配置改为false消除事件（会增加operatorgroup受管ns的CSV数量）
