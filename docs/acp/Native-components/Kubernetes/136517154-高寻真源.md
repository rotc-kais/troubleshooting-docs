# 高寻真源

新加 worker 节点上的 pod 访问不通

## Cause
- 节点 IP 地址冲突（原 1.95 改为 1.96）导致 ovs-ovn 组件状态异常

## Resolution
- 删除 chassis 记录，重建 ovs-ovn 和 kube-ovn-cni pod
