# tke

tke-platform-controller报证书未知授权(unknow authority)

## Cause
- 证书与master节点内容不一致
- sentry服务被停止导致证书同步机制失效

## Resolution
- 更新sentry-cm中的证书内容
- 重新开启sentry服务
