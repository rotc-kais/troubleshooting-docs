# Pod 无法访问集群外网络

Pod 无法访问集群外网络 ovn0 网卡未收到来自 Pod 的流量 OVN 路由表中未找到业务 Pod IP 对应的路由

## Cause
- etcd 异常导致 Kube-OVN 无法正确获取数据
- OVN 路由缺失导致流量无法正确转发

## Resolution
- 恢复 etcd 和 apiserver 服务
- 删除并重建受影响的 Pod
