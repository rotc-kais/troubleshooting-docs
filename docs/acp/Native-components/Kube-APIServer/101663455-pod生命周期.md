# pod生命周期

JIRA Jira：

## Cause

## Resolution
- 节点宕机后40秒变为NotReady状态
- 节点NotReady后30秒驱逐Pod，检查/etc/kubernetes/manifests/kube-apiserver.yaml中的--default-not-ready-toleration-seconds=30和--default-unreachable-toleration-seconds=30参数
