# 云平台故障告警

pod频繁健康检查失败发生重启 磁盘io升高 kube-apiserver日志存在etcd server timeout超时信息

## Cause
- 系统盘在周末存在定时任务导致写操作特别慢

## Resolution
- delete pod修复metis pod
- 执行crictl rm -f重启kube-apiserver、kube-controller、kube-scheduler
- 建议客户向iaas层确认系统盘定时任务
