# 删除pod后pod无法新建

删除pod后pod无法新建 apiservice报证书错误

## Cause
- kube-apiserver内部交付的证书有效期1年到期

## Resolution
- 检查kube-apiserver容器运行时间，超过1年时手动删除重建
