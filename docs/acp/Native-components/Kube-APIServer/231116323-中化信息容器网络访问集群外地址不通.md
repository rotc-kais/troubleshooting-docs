# 中化信息，容器网络访问集群外地址不通

容器无法访问集群外地址10.156.154.194:5236 节点10.6.158.253可访问该地址 其他节点无此问题

## Cause
- 物理网卡抓包无数据
- 节点iptables规则与其他节点一致但问题存在

## Resolution
- 联系iaas团队进行底层网络排查
