# 灾备环境平台组件重启原因排查

metis组件重启前存在报错 apiserver组件对应时间段出现异常日志

## Cause
- apiserver连接etcd不稳定导致接口调用异常

## Resolution
- 属于apiserver与etcd连接不稳定的偶发现象，后续自动恢复
