# 新建集群ovn

ovn-central无法启动 日志中存在连接127.0.0.1:6642失败的报错 netstat显示apiserver占用6642端口

## Cause
- apiserver进程通过随机端口占用6642端口导致端口冲突

## Resolution
- 重启apiserver释放端口占用
- 等待ovn-server重新监听6642端口
