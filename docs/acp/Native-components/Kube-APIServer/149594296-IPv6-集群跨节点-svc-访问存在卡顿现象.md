# IPv6 集群，跨节点 svc 访问，存在卡顿现象

创建流水线时访问webhook超时失败 跨节点访问SVC IP+Port首次正常后续卡顿 ovn0网卡因cksum error导致SYN报文被丢弃

## Cause
- geneve_sys_6081接口校验和错误导致TCP SYN报文被丢弃

## Resolution
- 执行 ethtool -K genev_sys_6081 tx off 关闭各节点校验
