# 3.6.1

业务应用需要使用40000以上的端口用于服务暴露

## Cause
- kube-apiserver默认nodeport端口范围未包含所需端口

## Resolution
- 修改所有集群master节点的/etc/kubernetes/manifests/kube-apiserver.yaml文件，在--service-cluster-ip-range下方添加- --service-node-port-range=xxxx-xxxx
- 重启kubelet服务
