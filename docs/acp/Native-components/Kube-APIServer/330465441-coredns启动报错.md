# coredns启动报错

Get "https://172.16.0.1:443/api/v1/services?limit=500&resourceVersion=0": dial tcp 172.16.0.1:443: i/o timeout 容器网络无法连通 apiserver

## Cause
- 子网 cidr 和 docker bridge 的网段冲突
- 虚拟机迁移导致底层网络迁移

## Resolution
- 对默认子网进行重新调整

## [workaround]
- 新增一个 ovn 子网段进行 IP 分配
