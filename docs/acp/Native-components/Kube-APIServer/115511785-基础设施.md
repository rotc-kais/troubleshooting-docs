# 基础设施

应用页面偶现无法加载，刷新可以正常显示 部署应用点击主机选择器无法加载选择项 平台应用页面创建应用按钮偶现无权限

## Cause
- global集群erebus服务存在bug
- nginx-ingress-controller服务存在异常pod

## Resolution
- 重启异常的nginx-ingress-controller服务
- 升级平台版本到3.0
