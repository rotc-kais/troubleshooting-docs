# 3.14

etcd无法启动 启动日志报peer相关集群配置异常

## Cause
- /etc/kubernetes/manifests/etcd.yaml中peer相关节点信息内容不全
- 历史手动操作etcd节点导致配置缺失

## Resolution
- 从正常节点获取缺失内容补充至etcd.yaml
- 手动触发apiservice重启
