# master节点异常宕机导致服务报错调用api

界面报错：调用kube-api-server失败 节点pod都处于删除中 controller-manager不工作，无法更新pod信息、ep信息

## Cause
- master节点etcd和controller-manager组件同时为leader时发生宕机

## Resolution
- 修复异常主机状态
- 重启该主机恢复
