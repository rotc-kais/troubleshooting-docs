# 手动删除备集群多余的etcd key

备集群中存在未被回收策略清理的etcd key key创建时间为3月22日

## Cause
- etcd回收策略未生效

## Resolution
- 确认etcd版本为3.16.2
- 使用etcdctl工具连接备集群etcd实例
- 执行删除命令: etcdctl del /path/to/redundant/key
- 重新巡检确认清理结果
