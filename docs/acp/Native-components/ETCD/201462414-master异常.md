# master异常

docker报错stream copy error- reading from a closed fifo master服务器OOM并CPU飙升后无响应 集群中存在14w个etcd-bak的pod

## Cause
- etcd-bak任务异常导致大量pod堆积，集群算力不足

## Resolution
- 通过etcdctl命令直接删除pods数据
- 检查各节点磁盘占用情况和etcd备份配置
