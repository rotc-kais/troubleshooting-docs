# 节点宕机导致ETCD备份的job反复执行重建

节点宕机后etcd备份任务持续创建大量Pod 界面删除备份任务后任务数量仍增加

## Cause
- master节点宕机导致etcd备份Job未被正确删除
- 控制器持续尝试重建Pod

## Resolution
- 手动删除对应的etcd备份Job：kubectl delete job <job-name>
