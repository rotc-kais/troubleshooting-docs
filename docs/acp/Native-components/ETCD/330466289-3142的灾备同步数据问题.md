# 3.14.2的灾备同步数据问题

在global集群插件处未看到etcd数据同步插件

## Cause
- 灾备同步数据通过后台部署的etcd-mirror实现，而非前台插件

## Resolution
- 确认数据同步由etcd-mirror后台服务处理
