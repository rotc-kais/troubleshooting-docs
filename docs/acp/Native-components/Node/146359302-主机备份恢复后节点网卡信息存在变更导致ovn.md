# 主机备份恢复后，节点网卡信息存在变更，导致ovn

ovn-cni启动失败 提示查询到错误网卡

## Cause
- 主机备份恢复后节点网卡信息变更

## Resolution
- kubectl edit node <nodename>设置正确的网卡信息
