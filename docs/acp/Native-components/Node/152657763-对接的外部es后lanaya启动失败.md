# 对接的外部es后lanaya启动失败

lanaya启动报错：no Elasticsearch node available

## Cause
- lanaya的deployment中配置了elasticsearch-basic-auth的secret，但对接的外部ES采用无认证方式

## Resolution
- 从lanaya的deployment中删除elasticsearch-basic-auth的secret配置
