# 界面报错未知错误

界面报错未知错误 health check timeout: no Elasticsearch node available

## Cause
- 密码加密方式错误

## Resolution
- 使用正确方式加密ES密码: echo -n es密码 |base64
- 更新 acp-config-secret 的 ES_PASSWORD 字段
- 更新 elasticsearch-basic-auth 的 password 字段
- 重启apollo服务并清理浏览器缓存
