# 计算组件

不支持多node挂载

## Cause
- pvc访问模式为单节点读写(RWO)
- sts配置指定了具体pvc名称导致3实例共用同一RWO模式pvc

## Resolution
- 修改sts存储配置，声明存储类并自动创建3个pvc
