# GPU服务忽然无法启动

Pod无法调度到节点 报错node not fit pod 节点GPU资源不足

## Cause
- 应用异常退出导致GPU资源未被释放

## Resolution
- 在目标节点执行GPU资源重置命令：nvidia-smi --gpu-reset
