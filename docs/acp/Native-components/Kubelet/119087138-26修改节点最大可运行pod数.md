# 2.6修改节点最大可运行pod数

节点最大pod运行数量默认为128个

## Cause
- kubelet默认配置限制

## Resolution
- 通过systemctl status kubelet查找kubelet.service文件路径
- 在EnvironmentFile路径中添加MAX_PODS="--max-pods=256"
- 在kubelet.service文件的ExecStart中添加$MAX_PODS环境变量
- 执行systemctl daemon-reload && systemctl restart kubelet

## [workaround]
