# 重启机器后kubelet无法启动

节点notready kubelet无法启动 ovn的pod显示running状态但节点无运行中容器

## Cause
- 残留的容器/未使用的文件导致资源冲突

## Resolution
- 执行 docker system prune -a 清理所有退出的容器和未使用的文件
