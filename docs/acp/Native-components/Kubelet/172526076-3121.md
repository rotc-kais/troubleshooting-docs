# 3.12.1

pod无法挂载pv kubelet日志报错：/etc/kubernetes/manifests: couldn't parse as pod(Object 'Kind' is missing in 'null')

## Cause
- /etc/kubernetes/manifests被错误修改为文件属性而非目录)

## Resolution
- rm -rf "/etc/kubernetes/manifests"
- mkdir /etc/kubernetes/manifests
