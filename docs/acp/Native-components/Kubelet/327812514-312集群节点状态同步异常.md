# 3.12集群节点状态同步异常

vip6443返回超时 三个master节点中两个显示异常

## Cause
- vip节点关机导致vip6443不可达
- kubelet.conf配置中的server地址未使用vip导致部分节点未报异常

## Resolution
- 修复vip节点确保vip可用
- 检查kubelet配置中server地址配置
