# Underlay 模式下节点重启后网络不通

节点重启后有概率网络不通

## Cause
- 节点网卡上的IP和路由未正确转移到OVS网桥

## Resolution
- 更新Kube-OVN到已修复该问题的新版本
- 通过挂载ConfigMap将修复后的start-ovs.sh脚本注入ovs-ovn Pod

## [workaround]
