# 3.4.2

容器组监控页面数据加载失败 Prometheus中缺少容器信息 异常节点上的容器监控数据缺失

## Cause
- kubelet的bug导致cAdvisor无法收集pod数据

## Resolution
- 重启异常节点kubelet服务
