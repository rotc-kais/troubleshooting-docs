# etcd 异常反复重启

etcd容器反复重启 报错"remote error: tls: bad certificate" 存在约11万etcd备份失败的Pod

## Cause
- etcd数据损坏

## Resolution
- 删除etcd数据清理
- 重启3个master节点的kube-scheduler组件
