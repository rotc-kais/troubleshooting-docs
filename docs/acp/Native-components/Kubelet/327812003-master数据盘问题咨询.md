# master数据盘问题咨询

sdb盘损坏

## Cause

## Resolution
- 备份/var/lib/docker、/cpaas目录数据到其他位置
- 停止docker和kubelet服务
- 卸载原挂载点
- 更换损坏的sdb盘
- 重新挂载新盘到/var/lib/docker和/cpaas
- 启动服务
