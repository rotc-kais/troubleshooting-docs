# 3.4.2

master1节点kubelet服务异常 kubelet.conf证书CN字段异常

## Cause
- kubelet.conf证书中CN字段被错误签发为system:master

## Resolution
- 执行证书更新脚本更新证书字段为system:node
- 修改kubelet.conf文件内容
- 重启kubelet服务
