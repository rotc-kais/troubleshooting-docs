# 更改Centos节点sem内核参数配置

容器内sem参数不生效

## Cause
- kernel.sem是命名空间级参数，需在Pod级别单独配置

## Resolution
- 修改/var/lib/kubelet/config.yaml添加allowedUnsafeSysctls配置
- 添加配置项：allowedUnsafeSysctls:
- kernel.sem
- 重启kubelet服务生效
- 在Pod配置中通过securityContext.sysctls设置参数
