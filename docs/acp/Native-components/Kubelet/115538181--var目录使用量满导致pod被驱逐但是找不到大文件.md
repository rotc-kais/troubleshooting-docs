# /var目录使用量满导致pod被驱逐，但是找不到大文件

/var目录使用量爆满但未找到大文件 Pod因磁盘空间被驱逐

## Cause
- 更换docker盘时未清理旧的/var/lib/docker数据

## Resolution
- 停止docker和kubelet服务
- umount /var/lib/docker目录
- 删除旧的/var/lib/docker/overlay2数据
- 重新挂载/var/lib/docker并启动docker、kubelet
