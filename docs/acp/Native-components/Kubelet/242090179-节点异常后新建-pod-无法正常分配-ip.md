# 节点异常后，新建 pod 无法正常分配 ip

新建 pod 无法正常分配 ip kube-ovn-controller leader 未调度到其他节点

## Cause
- kube-ovn-controller leader 所在节点故障且未触发重新选举

## Resolution
- 修改 kube-ovn-controller 添加 node selector 固定到 master 节点

## [workaround]
- 重启故障节点
