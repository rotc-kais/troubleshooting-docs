# Coredns 解析异常

域名解析结果不正确（例如：https://oss-tianjin.sangforcloud.com 解析到异常IP 118.26.48.245） 存在其他DNS解析相关异常

## Cause
- 节点resolv.conf文件中存在search域配置

## Resolution
- 检查所有节点的/etc/resolv.conf文件
- 注释或删除search域配置项
- 逐个重启coredns容器
