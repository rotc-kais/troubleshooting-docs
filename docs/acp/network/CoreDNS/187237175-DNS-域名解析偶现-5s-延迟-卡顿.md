# DNS 域名解析偶现 5s 延迟/卡顿

Pod 中访问域名时，偶现 5s 延迟/卡顿 kube-ovn-pinger 中无此问题

## Cause
- debian-slim:10 base 镜像版本存在 DNS 解析相关 bug

## Resolution
- 更新 base 镜像至 debian 11 或 12 版本
