# Pod 无法访问集群外部域名

Pod 无法解析外部域名，但节点能够正常解析

## Cause
- 节点上配置的 DNS 服务器与 CoreDNS 使用的 upstream 不一致，修改节点 DNS 后未重建 CoreDNS

## Resolution
- 重建 CoreDNS
