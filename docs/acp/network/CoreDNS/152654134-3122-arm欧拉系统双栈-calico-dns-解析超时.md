# 3.12.2 arm，欧拉系统，双栈 calico dns 解析超时

DNS解析超时 请求未到达CoreDNS Pod UDP报文存在checksum error计数

## Cause
- Calico vxlan.calico网卡UDP校验和问题

## Resolution
- 关闭vxlan.calico网卡的UDP校验和校验功能
