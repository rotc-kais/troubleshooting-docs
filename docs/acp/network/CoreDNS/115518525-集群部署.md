# 集群部署

coredns启动失败 kube-ovn-pinger启动失败

## Cause
- 麒麟OS物理机默认安装的安全插件导致OVN组件启动失败

## Resolution
- 登录节点执行检查命令：ps -ef |grep security
- 联系厂商确认后卸载安全插件
