# 3.4.2

nevermore一直重启，健康检查不通过 审计事件日志无呈现 容器内curl无法解析域名

## Cause
- coredns的cm文件hosts配置缺少fallthrough参数

## Resolution
- 修改coredns的cm文件添加fallthrough配置
- 重启coredns服务
