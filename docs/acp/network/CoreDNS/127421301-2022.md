# 2022

dns 解析失败 pod 的 /etc/resolve.conf 存在 cluster.local localdomain 配置

## Cause
- 节点 /etc/resolve.conf 存在多余 search localdomain 配置
- pod 继承错误 search 域配置

## Resolution
- 删除节点 /etc/resolve.conf 的多余 search localdomain 配置
- 验证 pod 的 /etc/resolve.conf 配置更新
