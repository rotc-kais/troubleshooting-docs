# 0

erebus组件无法启动 日志报错域名解析失败

## Cause
- 域名解析失败

## Resolution
- 编辑coredns的configmap: kubectl edit cm -n kube-system coredns
- 增加hosts内容: hosts { 192.168.0.0 aaa.bbb.com fallthrough }
- 重启coredns组件
- 重启erebus组件
