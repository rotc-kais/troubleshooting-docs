# 集群coredns解析错误率超过5%问题排查

coredns解析错误率超过5% 响应码为refused kube-ovn-pinger组件pod发起的查询请求

## Cause
- kube-ovn-pinger组件尝试解析alauda.cn时被拒绝

## Resolution
- 在coredns配置中增加hosts解析：alauda.cn -> 1.1.1.1
