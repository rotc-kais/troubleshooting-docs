# coredns解析不同二级域的方式

需要解析不同二级域名到不同上游DNS服务器 使用hosts配置方式无法应对多域名和IP变更场景

## Cause
- hosts配置方式无法动态适应多域名和IP变更需求

## Resolution
- 配置Corefile多DNS区块
- 为不同域名配置独立forward规则
