# ovs

Pod无法解析域名但可通过Service IP访问 ovs-vswitchd日志报错: ofproto_dpif_xlate: received packet on unknown port ovs UDP conntrack存在异常源IP记录

## Cause
- ovs-ovn组件状态异常导致无效端口映射
- 连接跟踪表存在异常条目

## Resolution
- 删除并重建故障节点上的ovs-ovn pod
