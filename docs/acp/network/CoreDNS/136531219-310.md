# 3.10

域名解析不通 hosts配置不生效

## Cause
- 3.10版本后coredns配置文件从coredns-cm变更为cpaas-coredns
- 旧配置未迁移到新ConfigMap

## Resolution
- 检查cpaas-coredns配置
- 将原有coredns-cm配置内容迁移到cpaas-coredns配置
