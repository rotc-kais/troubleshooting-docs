# 3.14.2

aml页面一直加载中，无法显示数据 请求返回发现访问gitlab有问题

## Cause
- 访问平台电脑未配置gitlab域名的hosts解析

## Resolution
- 在访问平台电脑的/etc/hosts中添加gitlab域名解析记录
