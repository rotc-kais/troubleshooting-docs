# 部署平台卡住

aiops/alauda-operators ars状态无法Ready sentry日志报错: failed to install chart for release describe artifacts显示无法解析平台域名

## Cause
- 容器内DNS解析失败
- 平台使用域名部署但未配置DNS

## Resolution
- 将平台域名解析配置到coredns的ConfigMap
- 重建coredns组件
- 等待组件自动恢复
