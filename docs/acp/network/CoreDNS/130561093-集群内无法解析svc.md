# 集群内无法解析svc

集群内的pod无法解析svcname，解析自己的svc也不行

## Cause
- 查看coredns没有任何日志，查看coredns的cm发现没有配置 fallthrough，这会导致如果 hosts 中找不到，则进入下一个 plugin 继续。缺少这一个指令，后面的 plugins 配置就无意义了；
