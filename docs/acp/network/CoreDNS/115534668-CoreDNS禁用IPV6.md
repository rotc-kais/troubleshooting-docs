# CoreDNS禁用IPV6

业务感知DNS解析慢

## Cause
- 容器内进程同时发起IPv4和IPv6解析请求，CoreDNS将无法解析的IPv6请求转发到上游导致延迟

## Resolution
- 修改CoreDNS ConfigMap，在Corefile中添加 `template ANY AAAA { rcode NXDOMAIN }` 配置
