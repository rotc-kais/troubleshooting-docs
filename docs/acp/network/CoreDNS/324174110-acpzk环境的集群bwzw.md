# acpzk环境的集群bwzw

创建命名空间失败 报错显示通过IPv6环回地址[::1]连接apiserver coreDNS部署失败但集群pod处于running状态

## Cause
- sysctl.conf文件中存在多余的IPv6配置
- 单栈IPv4集群错误尝试使用IPv6环回地址连接apiserver

## Resolution
- 删除三个master节点sysctl.conf中多余的IPv6配置
- 将sysctl.conf内容与标准部署文档中的IPv6配置对齐
