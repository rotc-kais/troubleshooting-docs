# 监控

grafana监控面板报错“cluster xxx not found” apollo组件日志报错无法解析svc域名

## Cause
- global集群主机dns配置存在search域导致apollo组件解析异常

## Resolution
- 注释dns search域配置
- 重启coredns和kube-proxy组件
