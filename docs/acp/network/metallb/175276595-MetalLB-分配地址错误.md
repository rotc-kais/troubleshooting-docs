# MetalLB 分配地址错误

Service分配的ip不在地址池内 MetalLB日志报错无法同步地址池 存在使用旧地址池的LoadBalancer类型Service

## Cause
- 旧地址池资源被删除后，仍存在使用该地址池的Service未被清理
- MetalLB会持续维护已分配地址池的Service直到全部删除

## Resolution
- 删除所有使用旧地址池的LoadBalancer类型Service
