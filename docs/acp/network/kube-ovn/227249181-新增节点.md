# 新增节点

kube-ovn-cni服务pod无法启动 ovn-controller日志报错同步10.200.55.57时出现错误 ovn-ovs日志报错'1 flows now, waiting for ovs-vswitchd flow ready'

## Cause
- 新增节点与master节点间6642端口通信不通

## Resolution
- 开通问题节点与集群master节点的6642端口通信
