# 容器网络异常无法连接外部网络

容器无法连接外部网络 容器IP可ping通 ovn-controller日志报错静态路由已存在

## Cause
- ovn 1.7.1的GC bug导致路由表未更新，pod-ip与ovn0绑定关系不符

## Resolution
- 升级ovn版本至1.8以上

## [workaround]
- 手动清理错误路由信息，触发ovn重新创建正确路由规则
