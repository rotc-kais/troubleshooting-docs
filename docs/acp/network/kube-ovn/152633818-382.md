# 3.8.2

ovn-pinger日志占用大量存储空间

## Cause
- ovn-pinger默认以5秒间隔探测网络状态导致高频日志输出

## Resolution
- 修改ovn-pinger部署参数，添加'--interval=60'将探测间隔调整为60秒
