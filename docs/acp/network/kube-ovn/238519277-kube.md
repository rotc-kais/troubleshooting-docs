# kube

pod 到节点的网络不通

## Cause
- 清理db后join子网的lsp无法自动恢复

## Resolution
- 通过`kubectl get ips`获取join子网ip
- 执行`ovn-nbctl lsp-add join node-192.168.140.12`和`ovn-nbctl lsp-set-addresses node-192.168.140.12 "{$mac} ${ip}"`手动创建lsp
