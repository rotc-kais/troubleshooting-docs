# kube

kube-ovn-cni pod持续crash重启 日志提示ovn0网卡不通

## Cause
- Logical Switch(ls)与Logical Router(lr)连接端口的MAC地址不一致

## Resolution
- 执行命令修改ls/lr端口MAC地址一致: kubectl ko nbctl lsp-set-addresses join-ovn-cluster 00:00:00:8A:9A:75
