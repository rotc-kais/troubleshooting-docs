# v3.12

pod无法分配ip underlay网卡ip测试和网关不通 ovn发出的报文带有vlan tag

## Cause
- 虚拟交换机(vswitch)网口配置了vlan id导致报文被双重封装

## Resolution
- 调整虚拟二层交换机配置，去掉vlan id
