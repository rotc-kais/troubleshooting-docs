# 健康检查偶现失败

健康检查偶现失败 空指针报错 超时

## Cause

## Resolution
- 手动将健康检查的exec修改为tcpSocket
- Kube-OVN后续版本将exec修改为httpGet

## [workaround]
- 修改健康检查方式为tcpSocket/httpGet并忽略tls eof报错
