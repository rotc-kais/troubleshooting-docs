# 从 ovn双网卡pod的第二块网卡丢失

创建的pod没有第二张网卡

## Cause
- /etc/cni/net.d/目录中同时存在00-kube-ovn.conflist和00-multus.conf导致CNI配置冲突

## Resolution
- 移除00-kube-ovn.conflist文件，重启multus daemonset和业务pod
