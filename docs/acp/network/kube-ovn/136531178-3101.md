# 3.10.1

kube-ovn-pinger和cert-manager组件无法启动 容器内ping其他节点的容器ip不通 节点间到容器ip通信正常

## Cause
- 主机网卡驱动问题导致校验失败

## Resolution
- 在所有节点执行命令: ethtool -K ETH tx off (ETH替换为实际物理网卡名)
