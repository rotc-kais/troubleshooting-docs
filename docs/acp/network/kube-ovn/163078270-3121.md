# 3.12.1

节点状态持续notready ovn-cni组件异常且不断重启 新增节点无法ping通ovn网关

## Cause
- 物理防火墙导致节点与ovn网关间网络不通

## Resolution
- 调整网络防火墙配置确保节点与ovn网关通信正常
