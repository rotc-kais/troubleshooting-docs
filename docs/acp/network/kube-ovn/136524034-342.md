# 3.4.2

添加节点失败，提示/etc/cni已存在 手动删除/etc/cni后界面未继续执行 再次清理并重新加入仍报错

## Cause
- 节点残留/etc/cni目录导致冲突

## Resolution
- 重新手动删除/etc/cni目录
- 在添加节点界面继续操作
