# kube

无法解析外部域名 pod访问外部ip不通 coredns日志出现io timeout

## Cause
- kube-ovn-controller同步subnet route policy时节点cache未完成同步

## Resolution
- 修改kube-ovn-controller代码逻辑，在处理节点update事件时触发子网route更新（https://github.com/kubeovn/kube-ovn/pull/5166）
