# kube

跨节点访问harbor-nginx pod时curl间断性不通 ping测试正常无丢包 抓包显示请求到达pod但无返回报文

## Cause
- harbor-nginx pod运行状态异常
- 应用层处理异常

## Resolution
- 检查harbor-nginx pod运行状态及配置
- 重启异常harbor-nginx pod

## [workaround]
- 使用相同镜像创建临时pod进行测试访问
