# 3.6.2

外网业务集群平台日志和事件不显示数据 tracer-agent容器内curl无法解析域名 underlay外网业务集群容器内无法ping通平台访问地址

## Cause
- 容器网段与nameserver ip之间存在网络策略限制
- underlay外网业务集群到平台访问地址存在路由问题

## Resolution
- 开通外网underlay业务集群容器网段与nameserver ip的网络策略
- 修复underlay外网业务集群到平台访问地址的路由问题
