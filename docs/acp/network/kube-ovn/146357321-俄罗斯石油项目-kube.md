# 俄罗斯石油项目 kube

kube-ovn相关组件无法启动 日志显示443端口访问异常 global节点请求被对端关闭

## Cause
- 卡巴斯基占用9780端口导致流量重定向异常

## Resolution
- 执行 systemctl 关闭卡巴斯基
