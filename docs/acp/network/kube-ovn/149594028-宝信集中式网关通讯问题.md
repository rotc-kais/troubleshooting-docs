# 宝信集中式网关通讯问题

ovn-default子网下的部分pod出现访问不了外部网络的情况

## Cause
- 开源社区提交导致子网切换到分布式网关时subnet匹配条件被删除，修改子网影响了其他子网下的pod

## Resolution

## [workaround]
- 从平台管理-网络管理-子网处更新ovn-default子网，删除activateGateway对应的IP地址
