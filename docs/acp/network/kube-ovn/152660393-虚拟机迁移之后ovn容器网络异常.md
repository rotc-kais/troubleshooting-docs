# 虚拟机迁移之后ovn容器网络异常

容器IP无法访问 ovs-ovn容器因健康检查失败重启 节点间ovn0网卡IP无法ping通

## Cause
- 内网限制UDP 6081协议

## Resolution
- 放开内网UDP 6081端口限制
