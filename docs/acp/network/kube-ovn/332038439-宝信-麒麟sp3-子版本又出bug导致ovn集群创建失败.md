# 宝信 麒麟sp3 子版本又出bug导致ovn集群创建失败。

创建集群失败，kube-ovn-pinger 起不来

## Cause
- 麒麟sp3内核版本4.19.90-52.49.v2207.ky10.x86_64存在datapath生成失败问题

## Resolution
- 降级内核版本到4.19.90-52.44.v2207.ky10.x86_64
