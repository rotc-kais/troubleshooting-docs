# Kube

Module name openvswitch not found

## Cause
- 节点迁移过程中安全加固或其他操作导致文件权限不正确

## Resolution
- chmod o+r /lib/modules/`uname -r`/modules.*
- kubectl -n kube-system delete po -l app=ovs
