# 容器内无法访问 Service

容器内无法访问 Service

## Cause
- 内核中 openvswitch conntrack 模块版本过低

## Resolution
- 4系列内核升级至4.14+版本或手动编译OVS内核模块
- CentOS7系统升级至3.10.0-957+版本
