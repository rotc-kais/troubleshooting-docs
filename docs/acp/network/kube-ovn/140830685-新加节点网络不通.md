# 新加节点网络不通

新加节点上的pod无法访问 ovs/ovn pod报错无法commit到ovnsb 节点间pod ip无法ping通

## Cause
- ovn组件与ovnsb数据库通信异常
- 节点网络配置异常

## Resolution
- 检查ovn-controller/ovn-northd等组件日志
- 验证ovnsb数据库连接状态
- 检查节点ovs网桥配置和vtep端口
- 必要时重启ovn相关组件

## [workaround]
- 临时移除问题节点

## [Related Information]
- Environment: TKE 3.10 with OVN
- ovs-vswitchd
- ovn-controller
- ovn-northd
- ovnsb数据库
- br-int网桥
