# Kube

ovn组件pinger异常 连接apiserver失败

## Cause
- 内核版本过高（49）
- 可能受checksum影响

## Resolution
- 将操作系统内核版本从49降级到44

## [workaround]
- 执行`ethtool -K eth0 tx off`并重启pinger（无效）
