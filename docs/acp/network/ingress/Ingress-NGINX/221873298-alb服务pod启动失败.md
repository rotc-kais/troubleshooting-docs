# alb服务pod启动失败

alb服务pod启动失败 alb2容器日志中加载'tcp-ingress'ingress规则时报错

## Cause
- 在cloudmp-cloudmp命名空间下创建了平台不支持的TCP类型ingress规则

## Resolution
- 删除冲突的tcp-ingress规则
