# 多活云tke关闭http访问方案咨询

需要关闭TKE集群的HTTP访问 当前配置允许HTTP访问，存在安全风险

## Cause
- Ingress控制器配置暴露了HTTP服务

## Resolution
- 删除或修改暴露HTTP服务的Ingress规则
- 禁用Ingress控制器
- 配置Ingress控制器仅允许HTTPS流量
