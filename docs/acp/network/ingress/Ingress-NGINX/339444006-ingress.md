# ingress

ingress-controller 部署状态异常

## Cause
- /etc/cni/net.d 存在旧版multis插件残留
- OVN插件存在其他辅助插件残留

## Resolution
- 删除/etc/cni/net.d目录下的残留文件
- 移除OVN插件上的其他辅助插件
- 执行热加载恢复操作
