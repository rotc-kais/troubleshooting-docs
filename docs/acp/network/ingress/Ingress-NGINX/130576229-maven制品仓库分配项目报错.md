# maven制品仓库分配项目报错

制品库分配项目报错 请求结构体不正确 停用tenant组件后故障消失

## Cause
- tenant组件修改了URL导致请求被错误路由或处理

## Resolution
- 停用tenant组件（移除相关ingress）
