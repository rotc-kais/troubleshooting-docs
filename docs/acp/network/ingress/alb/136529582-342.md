# 3.4.2

alb暴露域名url长度过长导致浏览器无法进行访问，url长度3000+

## Cause
- alb的large_client_header_buffers参数设置为4 16K导致缓冲区不足

## Resolution
- 修改alb的large_client_header_buffers参数为4 32K
- 执行kubectl edit cm -n cpaas-system "alb文件"修改对应配置
- 重启alb容器使配置生效
