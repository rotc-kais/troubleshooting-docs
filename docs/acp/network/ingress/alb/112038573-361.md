# 3.6.1

alb流量很低 cpu负载很高 alb内的error日志数据量很大

## Cause
- 后端服务端口不可达导致alb无限重试
- 后端服务异常或端口配置错误

## Resolution
- 升级到3.7版本(修复了无限重试机制)
- 在configmap配置超时时间(默认5s)
- 为后端服务配置健康检查机制
