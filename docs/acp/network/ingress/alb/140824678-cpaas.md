# cpaas

alb-cpaas-system pod无法启动 6443端口被占用报错

## Cause
- 存在gateway路由规则占用了6443端口

## Resolution
- 1. 执行 kubectl get gateway -A 查询占用端口的规则
- 2. 删除占用6443端口的gateway路由规则
- 3. 重启alb pod
