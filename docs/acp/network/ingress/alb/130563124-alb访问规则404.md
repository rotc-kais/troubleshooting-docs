# alb访问规则404

访问ALB规则返回404 直接访问SVC和EP地址+端口返回正确内容 应用存在/webapi接口但ALB返回不一致

## Cause
- ALB路径规则优先级配置不当
- /webapi和/路径规则优先级相同导致错误匹配

## Resolution
- 调整/webapi规则的优先级大于/路径规则
