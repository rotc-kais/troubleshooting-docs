# 集群使用 Pod IP 对外暴露方案后，阿里云 SLB 不能正常工作

非 metallb vip 节点上的 Pod 作为 client 无法正常访问 SLB metallb vip 节点能正常访问

## Cause
- SLB 可能只做了 DNAT 没有做 SNAT
- 与 SLB 的实现方式有关

## Resolution
- 联系阿里云进行处理
