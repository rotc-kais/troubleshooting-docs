# alb支持配置自定义CORS 跨域

JIRA Jira：

## Cause

## Resolution
- kubectl get rule -n $NS
- kubectl edit rule -n $NS $rulename
- 在spec下添加/修改字段：
  enableCORS: true
  corsAllowHeaders: a,b,c,d
  corsAllowOrigin: baidu.com
