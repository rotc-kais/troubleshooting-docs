# 集群alb故障

alb组件被驱逐 alb nginx容器启动异常

## Cause
- /目录磁盘使用超过85%触发kubelet驱逐
- alb配置文件proxy_buffer配置异常

## Resolution
- 还原默认配置
- 更新alb2资源
- 重启alb组件
