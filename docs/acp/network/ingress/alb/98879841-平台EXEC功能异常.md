## Resolution

- 3. 咨询客户使用浏览器版本，感觉部署要求使用适配谷歌浏览器进行测试，发现问题同样存在；
- 4. 客户反馈访问有使用代理且没有开启websocket的支持
- ## 解决方案
- 客户将代理开启websocket的支持后，EXEC功能连接恢复
