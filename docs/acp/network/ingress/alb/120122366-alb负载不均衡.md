# alb负载不均衡

并发10个请求时，个别pod接收了2个请求

## Cause
- alb负载均衡采用概率均衡机制，不同nginx process处理请求导致轮询分配不均

## Resolution
- 不建议修改nginx work process配置，长期测试统计请求比例评估均衡性

## [workaround]
- 理论上设置nginx配置`work_process 1`可缓解，但会影响性能评估
