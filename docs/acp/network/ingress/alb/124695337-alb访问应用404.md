# alb访问应用404

alb访问应用返回404 请求被转发到错误的端口9090而非正确的8088

## Cause
- 存在优先级更高的url为/的规则，导致/webapi请求被错误路由

## Resolution
- 调整/webapi规则的优先级高于/规则
