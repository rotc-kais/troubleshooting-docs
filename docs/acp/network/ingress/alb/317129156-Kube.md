# Kube

通过ALB的访问无法访问Ingress配置的域名 在集群节点和浏览器均无法访问Ingress域名

## Cause
- ALB使用hostNetwork模式导致NetworkPolicy的namespace白名单机制失效

## Resolution
- 在NetworkPolicy的ipBlock字段添加ALB节点IP白名单：
from:
  - ipBlock:
    cidr: 10.91.174.169/32

## [workaround]
