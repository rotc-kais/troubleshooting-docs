# alb的https规则被自动删除

https规则被自动删除 审计日志无删除记录 alb的debug日志显示脑裂

## Cause
- etcd存储性能差导致异常
- alb发生脑裂

## Resolution
- 迁移etcd存储

## [workaround]
- 重启alb实例
