# 通过nginx+alb访问页面缓慢

间歇性加载超时 TCP PSH包未到达nginx节点 路由缓存显示mtu 552

## Cause
- net.ipv4.ip_no_pmtu_disc=1导致PMTU发现被禁用
- 路由mtu设置异常

## Resolution
- 设置sysctl net.ipv4.ip_no_pmtu_disc=0
- 临时移除故障节点

## [workaround]
- 踢除故障节点
