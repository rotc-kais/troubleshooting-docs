# 平台页面菜单不显示

平台界面菜单不显示 浏览器访问HTTP接口显示network_connection_reset

## Cause
- 负载均衡器(如F5)对HTTP明文传输协议存在网络策略限制

## Resolution
- 检查global-alb负载均衡器针对HTTP协议的网络策略配置
- 改用HTTPS协议访问平台

## [workaround]
- 通过HTTPS协议访问平台
