# 中信银行

业务Pod的IP地址与节点tunl0网卡IP冲突（均为28.7.73.12） ALB负载不均衡导致其中一个Pod未收到请求

## Cause
- 异常节点的IPv4IPIPTunnelAddr注解错误指定了28.7.73.12地址，与Pod IP冲突

## Resolution
- 修改异常节点的IPv4IPIPTunnelAddr注解值为Calico分配的28.7.72.41地址
