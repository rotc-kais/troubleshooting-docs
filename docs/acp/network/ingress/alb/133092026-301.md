# 3.0.1

项目无法加载 浏览器控制台报错404

## Cause
- global集群master节点不稳定
- global-alb服务频繁重启
- alb规则同步期间规则丢失
- 3.0.1版本已知问题

## Resolution
- kubectl get pod -ncpaas-system |grep alb
- kubectl delete pod -ncpaas-system <albpod名称>
