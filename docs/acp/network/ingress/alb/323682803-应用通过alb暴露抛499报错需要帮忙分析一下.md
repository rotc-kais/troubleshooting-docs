# 应用通过alb暴露，抛499报错，需要帮忙分析一下

应用通过alb暴露时抛出499错误 alb日志中记录了客户端的podIP

## Cause
- 客户端在服务器处理请求前主动关闭了连接（如用户刷新页面、关闭标签页、网络中断等）

## Resolution
- 检查业务应用处理逻辑，排查响应时间过长或客户端提前终止请求的情况
