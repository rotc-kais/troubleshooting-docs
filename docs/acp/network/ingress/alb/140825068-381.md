# 3.8.1

修改业务alb对应的configmap配置文件内容后，重启captain组件会导致修改被还原

## Cause
- captain组件管理alb对应的helmrequest资源，重启captain会重置alb chart内的配置

## Resolution
- 避免直接修改configmap，应通过helm管理alb配置变更
