# 3.12.2

haproxy报错503，后端服务不可用 alb的nginx容器报错证书文件缺失 alb容器日志显示证书secret名称不匹配

## Cause
- haproxy使用的证书secret名称(ssl-2023)与alb服务期望的证书名称(2023-ssl)不一致

## Resolution
- 通过管理界面更新443端口使用的证书，保持两边secret名称一致
