# alb跨域报错

alb跨域报错

## Cause
- UI界面限制无法通过界面添加跨域配置
- ALB规则配置中缺少corsAllowOrigin设置

## Resolution
- 修改对应ALB的rule配置，在YAML的spec下添加corsAllowOrigin: http://158.245.4.246:8080
