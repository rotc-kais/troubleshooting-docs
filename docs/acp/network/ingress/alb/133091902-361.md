# 3.6.1

业务服务通过alb访问中断 kubelet事件显示Attempting to reclaim ephemeral-storage NodeHasDiskPressure状态告警

## Cause
- 业务服务容器未开放对应端口
- alb持续连接失败导致error日志堆积
- error日志占满ephemeral-storage触发驱逐

## Resolution
- 检查并修复业务服务端口配置
- 确认容器实际监听的端口与alb配置一致
