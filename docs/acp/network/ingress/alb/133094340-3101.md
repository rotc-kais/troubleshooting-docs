# 3.10.1

在alb下创建TCP 3306端口后主机未监听 重启alb组件后所有端口均未监听 alb日志显示1936端口占用

## Cause
- 客户创建的1936端口与alb指标端口冲突

## Resolution
- 删除客户创建的1936端口
