# alb无法转发至后端服务

无法访问 alb2 error.log中有报错no balancer found for

## Cause
- 后端服务svc.type使用了LoadBalancer类型
- alb设计不支持LoadBalancer类型服务

## Resolution
- 将svc.type修改为ClusterIP
