# alb无法访问后端服务

alb无法访问后端服务 alb2容器oom重启 错误日志：upstream timed out (110: Operation timed out) while connecting to upstream

## Cause
- alb2容器同步12000条负载均衡器规则导致内存不足

## Resolution
- 扩容alb实例内alb2容器的内存limit到4Gi
