# 更换业务域名证书后访问502

业务无法正常访问（502） policy中丢失访问规则的backend信息

## Cause
- 多个namespace下服务使用相同域名导致证书冲突
- 不同namespace的rule定义相同domain但使用不同证书

## Resolution
- 所有访问规则使用的证书绑定同一namespace下的保密字典
