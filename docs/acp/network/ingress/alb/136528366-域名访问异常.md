# 域名访问异常

通过域名访问到的服务与实际配置不符 svc直接请求服务正常 ALB的nginx日志显示域名请求被路由到同集群另一服务

## Cause
- 新建的负载均衡规则优先级(6)低于现有规则(5)
- 高优先级(5)规则配置了全路径匹配(/)

## Resolution
- 将新建规则的优先级从6调整为5
