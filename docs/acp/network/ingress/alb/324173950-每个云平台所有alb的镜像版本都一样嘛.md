# 每个云平台所有alb的镜像版本都一样嘛？

ALB镜像版本在不同云平台存在不一致

## Cause
- 客户自行更换了更高版本的ALB镜像

## Resolution
- 通过kubectl检查实际运行的ALB镜像版本
- 保持各平台ALB镜像版本一致性
