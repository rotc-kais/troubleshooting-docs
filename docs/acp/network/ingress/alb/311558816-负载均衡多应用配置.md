# 负载均衡多应用配置

无法将第二个应用的根路径调整为 baidu.com/test 多子路径（如 /test1/test2/test3）未正确映射

## Cause
- 需要为同一域名配置多个应用路径映射
- 未正确配置正则表达式捕获和路径重写规则

## Resolution
- 配置负载均衡路由规则：类型=域名，参数=baidu.com，URL匹配方式=RegEx，正则表达式=^/test(/|$)(.*)，重写目标=/$2
- 扩展多应用路径规则：正则表达式=^/law(/test[1-3])(/|$)(.*)，重写目标=/$1$3
