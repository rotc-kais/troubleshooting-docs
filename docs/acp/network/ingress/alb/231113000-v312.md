# v3.12

生产环境业务服务访问异常，没有请求到对应的服务 nginx日志出现错误信息：upstream sent invalid header: "\x20..." while reading response header from upstream

## Cause
- 业务服务跨域配置(CORS)响应头包含非法字符(空格)

## Resolution
- 修正业务服务跨域配置，去除响应头中的非法空格字符
