# 创建alb失败

创建alb失败 helmrequest资源报错已存在 删除hr资源失败，报错无法连通业务集群vip:6443

## Cause
- 存在pending状态的同名alb hr资源
- global集群无法连通业务集群vip:6443端口

## Resolution
- 开通global集群到业务集群vip 6443端口的网络转发
