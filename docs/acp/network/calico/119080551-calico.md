# calico

calico-node无法启动 监控检查失败 应用pod调度到节点后导致calico-node无法启动

## Cause
- 应用抢占9099端口导致端口冲突

## Resolution
- 修改应用yaml文件端口配置为其他未占用端口(如9098)
- 重新部署应用并检查calico-node状态(kubectl get pods -n kube-system)
