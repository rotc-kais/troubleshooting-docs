# calico

calico-node无法启动 日志报错127.0.0.1:9099: bind: address already in use

## Cause
- calico-node历史进程占用9099端口

## Resolution
- 执行ps -aux |grep calico检查历史进程
- 使用kill -9 进程号清理历史进程
