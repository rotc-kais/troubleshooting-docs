# 容器网络（Calico）

跨节点容器通信失败 网络性能低于预期 tunl0 隧道设备存在/缺失异常

## Cause
- IPIP 模式封包/解包导致性能损耗
- BGP 对等体未正确建立
- 路由表未正确传播

## Resolution
- 切换网络模式：calico_backend: bird"（BGP）或 "ipip"
- 检查 BGP 配置：/etc/calico/confd/config/bird.cfg
- 验证路由传播：calicoctl node status
