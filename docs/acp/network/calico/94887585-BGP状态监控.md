# BGP状态监控

BGP状态未处于Established状态 Global Peer被删除 节点处于NotReady状态

## Cause
- acp2.6版本calico未暴露metric指标

## Resolution
- 部署监控脚本到所有节点
- 配置定时任务执行脚本
- 调整脚本中GLOBALPEERNUM值为实际peer数量
- 对接监控平台webhook地址

## [workaround]
- 通过calicoctl node status手动检查BGP状态
