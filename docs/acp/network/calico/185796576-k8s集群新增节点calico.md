# k8s集群新增节点calico

calico-node使用本地网络启动 podip在同一网段

## Cause
- 节点名称解析到错误网卡IP

## Resolution
- 重新设置节点hosts解析到正确网卡IP
- 删除重建calico-node服务pod
- 重启使用本地节点网络的服务
