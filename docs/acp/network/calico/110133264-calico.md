# calico

calico-node无法启动处于0/1状态 Pod事件显示网卡异常 检测到的IP与预期网卡(eth0)不匹配

## Cause
- Calico-node启动时检测到错误的IP地址(节点PodIP)
- 检测到的IP与配置的网卡不匹配

## Resolution
- 通过重启calico-node及对应IP的服务恢复
