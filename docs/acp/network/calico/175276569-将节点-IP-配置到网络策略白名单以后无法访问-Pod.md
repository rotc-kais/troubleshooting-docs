# 将节点 IP 配置到网络策略白名单以后，无法访问 Pod

无法访问其他节点上的 Pod

## Cause
- 访问使用了 tunnel IP（tunl0/vxlan.calico 的 IP）而非节点 IP

## Resolution
- 将 tunnel IP 加入网络策略白名单
- Kube-OVN 需同时将 join 网段和节点 IP 加入白名单
