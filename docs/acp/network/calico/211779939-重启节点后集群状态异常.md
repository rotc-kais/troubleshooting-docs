# 重启节点后集群状态异常

calico-node组件异常 认证失败错误日志 calico-kubeconfig文件token签发时间与当前时间不匹配

## Cause
- master3节点主机时间与其他master节点不一致

## Resolution
- 调整所有节点时间同步
