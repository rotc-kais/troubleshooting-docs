# calico

calico-node 异常，日志出现 bird: Netlink: Operation not permitted 以及 ipset/iptables Operation not permitted 报错 calico-node 容器的 capabilities 为空

## Cause
- 挂载在 /var/lib/containerd 的 LV 卷/XFS 文件系统损坏导致容器文件系统的 linux capabilities 属性缺失

## Resolution
- 修复存储设备/文件系统故障
- 删除已有的 calico-node 镜像并删除正常运行的 calico-node Pod
