# 部分calico

calico-node容器事件中频繁出现readiness健康检查失败 健康检查失败固定在特定节点出现，日志无明显错误，重启无效 手动执行健康检查命令无异常

## Cause
- 异常节点主机负载过高（300+），导致kubelet执行健康检查超时

## Resolution
- 调整健康检查超时时间到10s
- 联系iaas团队解决虚拟机高负载问题

## [workaround]
- 临时增大健康检查超时时间
