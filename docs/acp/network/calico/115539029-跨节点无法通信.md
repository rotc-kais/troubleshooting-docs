# 跨节点无法通信

跨 node 的 pod 与 pod 之间无法通信 node 与 node 之间无法通信

## Cause
- ipipMode 被配置为 Always 时，ipip 内核模块未正确加载或存在冲突

## Resolution
- 执行 `modprobe -r ipip` 命令卸载 ipip 内核模块

## [workaround]
