# OVN

Pod 无法 ping 通 Service IP 用户使用 ping 检查 Service 连通性

## Cause
- Underlay 模式流量不经过节点主机网络
- Kubernetes Service 不实现 ICMP 协议

## Resolution
- 使用正确方法检查 Service 连通性（如 curl 或应用层访问）
