# 3.6.1

项目和命名空间无法进行GPU配额 GPU配额不显示

## Cause
- 缺少cf-crl-gpu-manager-vcuda-core和cf-crl-gpu-manager-vcuda-memory的configmap资源

## Resolution
- 手动创建cf-crl-gpu-manager-vcuda-core和cf-crl-gpu-manager-vcuda-memory的configmap资源
