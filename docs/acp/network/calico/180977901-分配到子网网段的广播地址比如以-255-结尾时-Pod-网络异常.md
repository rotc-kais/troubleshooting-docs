# 分配到子网网段的广播地址（比如以 255 结尾）时 Pod 网络异常

Pod 网络异常

## Cause
- 交换机未转发网络包

## Resolution
- 将 network/broadcast 地址添加到保留 IP 中
- 修改交换机配置

## [workaround]
