# calico日志量很大，导致k8s日志量占用很大

calico日志量很大，导致k8s日志量占用很大 calico日志报错failed to disable XDP，failed to load BPF program

## Cause
- calico自动开启了XDP功能

## Resolution
- 在calico-node的ds和pod的yaml里添加环境变量FELIX_XDPENABLED=false关闭XDP功能
