# 虚拟机重启后容器网络断开，calico

虚拟机重启后容器网络断开 calico-node 0/1 状态异常

## Cause
- IaaS层异常导致虚拟机强制重启
- 基础镜像损坏导致内核调用错误

## Resolution
- systemctl stop kubelet
- docker rm -f <container_id>
- docker rmi <image_id>
- systemctl start kubelet
- kubectl get pods -n kube-system | grep calico-node
