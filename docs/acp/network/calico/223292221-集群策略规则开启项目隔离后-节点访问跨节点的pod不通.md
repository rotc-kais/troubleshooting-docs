# 集群策略规则开启项目隔离后 节点访问跨节点的pod不通

节点访问跨节点的pod不通 节点访问podip时srcip是tunl0的ip

## Cause
- tunl0的ip属于calico pod cidr范围，触发namespace networkpolicy过滤规则

## Resolution
- 不建议在calico集群下开启集群策略项目隔离
