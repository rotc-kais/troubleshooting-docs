# 3.8.1

部署global失败 calico controller和calico-node pod无法正常运行 安装日志显示填写的网卡为eth0而实际网卡名称不符

## Cause
- 部署时填写的网卡名称（eth0）与主机实际网卡名称不匹配

## Resolution
- 所有部署节点执行清理脚本后重新部署，填写正确的网卡名称
