# calico组件运行异常

calico-node日志报错：Connection to the datastore is unauthorized error=connection is unauthorized: Unauthorized Pod资源AGE信息显示<invalid>

## Cause
- 主机时间不同步且同步后未重启kubelet及docker服务（时间差达2小时）

## Resolution
- 重启kubelet和docker服务

## [workaround]
