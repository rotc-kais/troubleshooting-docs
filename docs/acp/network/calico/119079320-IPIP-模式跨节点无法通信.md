  value: "true"- 执行 kubectl edit felixconfiguration default 添加 ipipEnabled=true:
apiVersion: crd.projectcalico.org/v1
kind: FelixConfiguration
metadata:
  name: default
spec:
  ipipEnabled: true## [workaround]- 手动执行 ip link set tunl0 up## [Related Information]- tunl0- calico-node- FELIX_IPINIPENABLED- FelixConfiguration- bird- ipipEnabled- Component: Calico- Page ID: 119079320- Original Title: IPIP 模式跨节点无法通信
