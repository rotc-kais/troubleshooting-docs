# 测试发版第7轮arm ipv6单栈环境出现nd异常

节点网络不通 ipv6的ip nd处于incomplete状态

## Cause
- net.ipv6.route.max_size参数默认值4096不足

## Resolution
- 执行sysctl net.ipv6.route.max_size=16384

## [workaround]
- 重启节点
- 临时禁止Pod调度到问题节点
