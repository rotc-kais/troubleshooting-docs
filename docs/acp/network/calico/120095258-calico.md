# calico

calico-node异常无法启动 flexvol-driver_calico容器日志显示Image not present /usr/local/bin/flexvol

## Cause
- calico-node镜像损坏

## Resolution
- systemctl stop kubelet
- docker images | grep calico
- docker rmi xxxx:60080/xx/xxx:xxx
- systemctl restart kubelet
