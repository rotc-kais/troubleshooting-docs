# 3.4.2

平台界面无法直接设置calico的ipip模式 默认ipip模式为Never不符合现场网络需求

## Cause
- 3.4.2版本平台界面未提供直接修改calico ipip模式的配置入口

## Resolution
- 通过集群-资源管理修改IPPool资源的ipipMode参数（注意大小写）
- 通过集群-资源管理修改Subnet资源的ipipMode参数（注意大小写）
- 修改后通过网络管理-网络检查验证容器网络状态
