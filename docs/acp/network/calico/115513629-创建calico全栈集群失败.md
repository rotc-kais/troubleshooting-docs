# 创建calico全栈集群失败

metis起不来，报错找不到secret: secret "metis-serving-cert" not found calico-node pod日志报错: bird: Netlink: Network is down, The new node cannot work

## Cause
- ipip模块导致网络异常

## Resolution
- 执行命令删除ipip模块: modprobe -r ipip
