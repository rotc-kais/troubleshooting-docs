# 2022

kubectl命令超时 新添加的master节点未被监控到 跨节点访问Pod网络不通

## Cause
- Calico的IP_AUTODETECTION_METHOD配置使用can-reach方式导致网卡选择异常
- 跨云环境（云上虚拟机+本地虚拟化）网络接口识别问题

## Resolution
- 修改Calico配置将IP检测方式改为指定接口：IP_AUTODETECTION_METHOD=interface=eth0
