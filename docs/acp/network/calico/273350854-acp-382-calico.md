# acp 3.8.2 calico

calico-kube-controller 内存飙升到 24G 报错 10.70.16.241 节点 not found

## Cause
- calico 历史 bug 导致不存在节点的 ipamblock 未清理

## Resolution
- 删除对应节点的 ipamblock 和 blockaffinities 并重启 calico-kube-controller
