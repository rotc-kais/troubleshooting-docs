# Calico

error getting IP from IPAM:Address already assigned in block pod describe事件信息无IP显示 calicoctl检查显示IP未被分配但实际已被占用

## Cause
- IPAM地址分配状态与实际使用情况不一致
- 调整副本时尝试分配已被占用的固定IP

## Resolution
- 移除服务固定IP配置并重启
- 服务正常启动后重新设置固定IP
