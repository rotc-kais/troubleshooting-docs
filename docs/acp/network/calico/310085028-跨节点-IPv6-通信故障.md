# 跨节点 IPv6 通信故障

NodePort 访问 50% 故障率 跨节点 IPv6 Pod IP 无法访问 tcpdump 显示 bad udp cksum

## Cause
- tx checksumming offload 导致 UDP 校验和错误

## Resolution
- 执行 ethtool -K eth0 tx off 关闭 TX checksum offload
