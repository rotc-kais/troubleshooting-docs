# pod 之间的 ping6 不通

pod ping6不通同节点pod 抓包未发现pod间ping6流量

## Cause
- ipv6 forwarding未启用(all接口)

## Resolution
- echo "1" > /proc/sys/net/ipv6/conf/all/forwarding

## [workaround]
