# v6数据包在传输网络中被丢弃

v6数据包在传输网络中被丢弃 数据包从node发出后未到达目标node

## Cause
- calico集群使用相同ASN
- 客户网络系统禁止同一AS内回环

## Resolution
- 每个node部署不同ASN
- 允许网络范围内nodes部署为同一AS
