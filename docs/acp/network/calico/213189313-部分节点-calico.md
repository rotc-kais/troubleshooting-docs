# 部分节点 calico

calico-node not Ready 文件权限报错

## Cause
- /run/xtables.lock owner不正确(非root)
- cloudwalker-agent systemd服务修改了文件权限

## Resolution
- chown root:root /run/xtables.lock
- chgrp root /run/xtables.lock

## [workaround]
