# calico

calico-node 状态为 Not Ready ipset 报错：kernel error received operation not permitted pod 中 iptables 命令报权限错误

## Cause
- containerd 目录所在文件系统类型为 ext4（正常节点为 XFS）

## Resolution
- 调整 containerd 目录所在文件系统的类型或挂载参数（如改用 XFS 文件系统）
