# 节点pod起不来显示calico 认证失败

节点上的容器处于ContainerCreating状态 calico认证失败

## Cause
- warpgate程序导致containerd透传env给cni程序
- cni组件错误使用warpgate的http代理

## Resolution
- 修复AIT插件warpgate的bug
