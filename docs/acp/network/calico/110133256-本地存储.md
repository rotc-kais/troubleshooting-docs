# 本地存储

服务磁盘使用率无法监控 节点数据为空 curl podip:8080/metrics |grep topolvm_volumegroup_available_byutes 无数据返回

## Cause
- topolvm-node所在节点的calico网络插件异常
- calico-node处于0/1未启动状态
- 网卡识别错误

## Resolution
- 检查topolvm-node所在节点的calico-node状态
- 重启calico服务
- 修复节点网络
