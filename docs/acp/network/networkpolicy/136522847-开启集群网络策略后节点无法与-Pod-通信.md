# 开启集群网络策略后节点无法与 Pod 通信

开启集群网络策略后节点间无法互ping对方节点上的Pod IP

## Cause
- ACL规则中残留旧join网段(200.102.0.0/24)配置

## Resolution
- 删除所有ACL规则
- 重启kube-ovn-controller使新ACL自动生成
