# 3.6.1

falco pod无法正常运行，一直crash 查看日志报错无法下载对应的内核模块

## Cause
- 部署安全组件falco时内核版本不满足要求，无法构建驱动

## Resolution
- 检查操作系统及内核版本是否符合支持列表要求
- 升级操作系统或内核到支持的版本
