# 3.8.1

部署TKE平台卡在镜像拷贝步骤 mctmp容器日志停止刷新 磁盘IO性能低下

## Cause
- 磁盘IO性能问题导致镜像拷贝进程阻塞

## Resolution
- 在三个节点执行docker restart重启minio容器
- 持续观察mctmp容器日志直至镜像拷贝完成
