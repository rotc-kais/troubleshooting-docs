# docker网段与宿主机冲突

网络故障 存在172.18.0.0网段的异常路由

## Cause
- 名为kind的Docker网桥(172.18.0.0/16)与宿主机新网段172.0.0.0/8冲突

## Resolution
- 删除kind网桥: docker network rm <network-id>
- 重启docker服务: systemctl restart docker
