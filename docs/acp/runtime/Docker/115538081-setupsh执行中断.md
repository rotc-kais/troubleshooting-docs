# setup.sh执行中断

执行setup.sh到安装docker步骤时vpn断线

## Cause
- vpn网段172.17.0.0/16与docker0默认网段冲突

## Resolution
- 修改docker0网段：在setup.sh的daemon.json中添加"bip": "192.168.200.1/24"

## [workaround]
