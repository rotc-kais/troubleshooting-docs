# 容器目录挂载咨询

使用ConfigMap或Secret挂载时Tomcat不会解压WAR包 容器内存在同名文件夹导致解压操作被取消

## Cause
- Tomcat检测到挂载的同名文件夹后取消解压操作
- 容器启动顺序为先挂载卷后执行解压操作
