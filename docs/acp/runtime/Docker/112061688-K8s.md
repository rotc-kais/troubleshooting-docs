# K8s

容器时间比宿主机时间早8小时

## Cause
- 容器默认采用UTC时区，宿主机采用CST时区

## Resolution
- 挂载宿主机时区文件到/etc/localtime
- 设置容器环境变量TZ=Asia/Shanghai
- 构建镜像时执行RUN rm -f /etc/localtime && ln -sv /usr/share/zoneinfo/Asia/Shanghai /etc/localtime && echo "Asia/Shanghai" > /etc/timezone
