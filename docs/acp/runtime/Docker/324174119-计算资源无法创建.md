# 计算资源无法创建

计算资源无法创建 etcd db size达到2.1G并出现空间不足告警

## Cause
- etcd数据库空间不足（默认2G限制）

## Resolution
- 执行etcd压缩及碎片整理
- 调整etcd db上限至4G
- 修改coredns deployment资源限制
- 调整operator csv资源配置
