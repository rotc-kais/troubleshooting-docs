# UDP 应用收包延迟

UDP应用偶现未收到回包导致重传，三次后报错 三次超时后日志显示同时收到三个回包，但抓包未发现对应现象 在Pod中复现问题，节点直接docker run无故障

## Cause
- Pod资源限制(2C 2G)导致应用处理延迟

## Resolution
- 调整资源限制至8C 16G

## [workaround]
- 临时调高Pod资源限制

- Environment: CentOS 7, Kubernetes with kube-ovn CNI underlay模式
