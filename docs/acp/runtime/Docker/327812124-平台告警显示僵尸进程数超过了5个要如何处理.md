# 平台告警显示：僵尸进程数超过了5个要如何处理

平台告警显示僵尸进程数超过5个

## Cause

## Resolution
- ps -ef | grep 进程号
- ps -ef | grep 父进程id
- docker inspect 容器id前五位 | grep Pid
- docker ps -a | grep 容器id前五位
- kill父进程使容器自动重启
- docker ps -a | grep 容器名称
- ps -e -o stat,ppid,pid,cmd |egrep '^[Zz]'
