# 集群异常

节点状态为NotReady kubelet运行异常 docker重启失败

## Cause
- /etc/docker/daemon.json文件存在语法错误（镜像仓库设置部分多出双引号）

## Resolution
- 修正/etc/docker/daemon.json配置错误
- 执行systemctl restart docker重启服务
