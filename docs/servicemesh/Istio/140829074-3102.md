# 3.10.2

无法在网关路由中选择网关 新建的网关只能分配给cpaas和acp两个项目 其他项目(demo/test)分配失败

## Cause
- 项目下缺少与项目同名的命名空间(ns)
- 客户启用了alpha功能导致创建项目时生成带cpaas-prj-前缀的ns

## Resolution
- 手动创建与项目同名的命名空间(ns)
