# 微服务无法启动,istio

微服务容器无法启动 istio-proxy报错dial tcp xxx:15012 i/o timeout 容器内无法连接svc地址及ep地址

## Cause
- istio组件和应用使用不同子网导致网络不互通

## Resolution
- 更新应用子网使其与istio组件处于相同子网
