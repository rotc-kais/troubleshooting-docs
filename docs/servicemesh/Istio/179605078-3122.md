# 3.12.2

ml-pipeline 相关 Pod 处于 1/2 就绪状态 sidecar.istio.io/inject 注解状态异常 istiod 日志存在异常

## Cause
- kubeflow 命名空间存在 istio-injection label 导致 sidecar 注入被阻止

## Resolution
- 执行 kubectl get ns kubeflow -o yaml 检查并删除 istio-injection label
