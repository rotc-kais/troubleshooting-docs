# 3.6

1. 排查发现asm组件运行正常，查看相关组件日志无明显报错信息；

## Cause
- 导致页面看不到相应的监控数据；

## Resolution
- 关键词：service mesh 流量监控
- service mesh配置完服务和请求连接后，不能正常监控调用链路
- ## 解决方案
- 通过服务网关请求微服务，新建分组绑定域名，域名解析ip为入口网关，配置API管理和服务路由后，在内网curl测试后监控数据显示正常，问题解决。如果微服务要对集群外提供访问，并且能通过拓扑、调用链、流量监控跟踪，
