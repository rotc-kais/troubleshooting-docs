# 业务容器pod日志报错

tail: unrecognized file system type 0x794c7630 for 'nohup.out'. reverting to polling

## Cause
- 基础镜像中的 coreutils 工具版本过低

## Resolution
- 重新获取新版基础镜像并重新打包应用镜像
