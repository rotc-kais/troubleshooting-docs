# 没有ping，无法安装ping

容器内无法使用ping命令 容器内无法安装ping命令

## Cause
- 容器使用的基础镜像未安装ping工具（如iputils包）

## Resolution
- 更换包含ping工具的基础镜像
- 在Dockerfile中安装iputils包
