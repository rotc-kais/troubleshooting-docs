# 创建业务集群失败

创建业务集群添加节点时检测到master节点连接拒绝

## Cause
- 节点主机名称包含下划线（不符合命名规范）

## Resolution
- 通过 hostnamectl set-hostname xxx 修改符合规范的主机名称
