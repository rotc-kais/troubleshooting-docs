# 平台健康状况显示异常

平台健康状况提示为故障

## Cause
- 平台健康状态检查了已停用的global集群es相关组件

## Resolution
- 升级到3.13版本获得优化

## [workaround]
- 忽略该健康状态告警（不影响实际功能）
