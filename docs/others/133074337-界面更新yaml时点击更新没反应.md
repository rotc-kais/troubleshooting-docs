# 界面更新yaml时点击更新没反应

界面更新yaml时点击更新没反应

## Cause
- 资源名称超过31位字符限制

## Resolution
- 规范资源命名使其符合前端31位长度限制

## [workaround]
- 通过后台直接修改资源配置
