# 更改监控策略，点击更新后，有弹窗告警

点击更新监控策略后出现弹窗告警 告警内容显示该告警already exists

## Cause
- 策略处于告警状态时不可更改

## Resolution
- 等待策略恢复告警状态解除后修改
- 关闭该策略后进行修改
