# 平台部署的mongo集群无法启动

界面卡在部署中 有状态副本集无容器组 事件显示资源问题报错

## Cause
- mongo-exporter资源限制未配置limits
- 命名空间默认limits导致requests超过限制

## Resolution
- 修改yaml手动补充mysql-exporter资源限制(requests/limits)
