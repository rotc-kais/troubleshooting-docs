# 3.12.1

global集群asm-istio-cacerts-业务集群名称证书时间小于30天预警

## Cause
- asm-istio-cacerts对应的secret内证书有效期仅2个月
- certificate资源设置的提前续签时间为15天

## Resolution
- cert-manager自动提前15天续签
