# 3.8.2

审计中存在很多alive-lock配置文件创建失败记录

## Cause
- 业务集群使用自建vip部署，alive-lock配置文件用于vip绑定检测

## Resolution
- 升级到3.11及以上版本（自建vip检测机制已变更）
