# 给容器配置host解析

JIRA Jira：

## Cause

## Resolution
- 使用hostAliases字段配置容器host解析，示例配置：

hostAliases:
- ip: "127.0.0.1"
  hostnames:
  - "foo.local"
  - "bar.local"
- ip: "10.1.2.3"
  hostnames:
  - "foo.remote"
  - "bar.remote"
