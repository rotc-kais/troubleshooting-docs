# demo环境部署推理服务失败

failed to find kserve-container in ServingRuntime containers

## Cause
- ClusterServingRuntime aml-mlserver-cuda-11.6 资源名称配置错误

## Resolution
- 从其他环境拷贝新的 aml-mlserver-cuda-11.6 资源并重新部署
