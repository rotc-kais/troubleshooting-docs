# 上架包异常

上架东方通operator后packageserver组件panic错误 panic: assignment to entry in nil map 删除下架后packageserver组件自愈

## Cause
- operator的csv文件中存在单词拼写错误

## Resolution
- 修正csv文件中的字段拼写错误

## [workaround]
- 删除问题operator包使packageserver恢复
