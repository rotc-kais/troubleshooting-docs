# pvc存储用量使用率问题

清理pvc中存储数据后存储用量使用率未有变化

## Cause
- pvc存储用量使用率是根据查询时间范围内取最大值的计算结果

## Resolution
- 调整查询时间范围或等待系统重新计算最大值
