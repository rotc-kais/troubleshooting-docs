# 调用链导致业务异常

istio-proxy oom 业务服务健康检查失败 pod重启

## Cause
- sidecar默认配置0.1核cpu过低
- 信创cpu环境资源适配问题
- 网格内微服务数量与流量影响资源消耗

## Resolution
- 调整sidecar资源配置为0.2核256Mi
