# 3.8、3.10修改日志保存时间超过30天

平台前端限制日志保存时间最多30天

## Cause
- 平台默认配置限制日志保存周期

## Resolution
- 通过kubectl edit修改logcenter模块的moduleinfoes资源，调整ttl字段值为期望天数
