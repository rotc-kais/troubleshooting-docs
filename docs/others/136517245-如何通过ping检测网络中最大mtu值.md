# 如何通过ping检测网络中最大mtu值

部分端口或url无法访问

## Cause
- cni的mtu和宿主机网卡mtu不一致导致大包丢弃

## Resolution
- ping -s 1500 ip
- ping -M do -s 32767 -c 4 <ip>
