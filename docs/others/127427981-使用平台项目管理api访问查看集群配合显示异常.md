# 使用平台项目管理api访问查看集群配合显示异常

部分集群的可分配额（allocatable）显示为负数

## Cause
- 项目配额（used）设置值大于集群实际可用资源（capacity）

## Resolution
- 调整项目配额配置使其小于等于集群实际资源
- 或扩容集群物理资源
