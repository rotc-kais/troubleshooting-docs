# ES索引如何配置合理的分片数和副本数

## 环境信息 适用版本：通用

## Cause

## Resolution
- 创建索引时设置分片和副本：PUT test { "settings":{ "number_of_shards":3, "number_of_replicas":0 } }
- 更新副本数：PUT test/_settings { "number_of_replicas":2 }
