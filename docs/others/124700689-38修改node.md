# 3.8修改node

需要修改node-exporter默认9100端口

## Cause
- 默认端口9100需要变更

## Resolution
- 执行命令修改DaemonSet配置：kubectl edit ds -n cpaas-system kube-prometheus-exporter-node
- 在DaemonSet中找到3处9100端口配置并修改为新端口
