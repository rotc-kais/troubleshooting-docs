# v3.12

global-asm-controller pod启动失败 pod持续crash，日志报错panic

## Cause
- ServiceMeshGroup资源内cluster缺少集群名称字段

## Resolution
- 手动在ServiceMeshGroup资源内cluster处添加集群名称，重启global-asm-controller pod
