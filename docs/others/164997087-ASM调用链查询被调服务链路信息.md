# ASM调用链查询被调服务链路信息

无法查询被调服务的下一跳链路信息 通过thread.id标签查询数据无结果

## Cause
- 平台3.12.2版本设计限制，选择被调服务时只能查询被调服务的tag

## Resolution
- 升级到3.14版本，该版本支持不输入主被调服务时查询任意span的tag
