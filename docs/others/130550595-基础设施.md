# 基础设施

每次页面打开看到的监控峰值不一致

## Cause
- 监控数据采集时间窗口动态变化
- 最新的数据点会随着当前时间向前移动
- 部分时间点数据为空导致显示异常

## Resolution
- 该现象属于平台监控正常采集逻辑设计预期行为，无需修复

## [workaround]
- 理解监控数据采集间隔计算公式：时间段/120个点
- 接受动态时间窗口导致的数据点显示差异
