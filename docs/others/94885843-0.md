# 0

pod下载镜像失败

## Cause
- cronjob配置的imagePullSecrets(dockercfg-region-test-test)在对应ns不存在

## Resolution
- 修改cronjob的imagePullSecrets为ns下已有的secret
- 创建同名secret(dockercfg-region-test-test)
