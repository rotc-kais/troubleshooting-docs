# 3.10.0

添加集群节点时报错检查超时，提示请重试 tke-platform-api服务报空指针错误并触发重启

## Cause
- 3.10.0版本无法识别4096位长度的SSH密钥格式

## Resolution
- 更换为密码登录方式
- 使用ssh-keygen -b 2048重新生成2048位密钥
