# 3.12.1

master节点nevermore组件pod一直触发OOM重启 节点cpaas/audit/data/diskqueue目录为10Gi

## Cause
- 历史压测操作导致缓存数据一直没有释放

## Resolution
- 删除节点cpaas/audit/data/diskqueue目录数据
