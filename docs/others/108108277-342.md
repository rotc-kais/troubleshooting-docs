# 3.4.2

部署平台一直卡在端口检查阶段 执行netstat和ss命令查看主机端口时命令长时间无返回（约5分钟）

## Cause
- 客户虚机模板存在问题

## Resolution
- 重启并更换主机模板后重新部署
