# 3.6.1

lanaya cpu使用率持续升高到90%以上 扩容cpu limit从2c到4c无效 日志中存在持续报错信息

## Cause
- 外部es存储磁盘空间已满
- es无法写入数据导致lanaya消费异常

## Resolution
- 清理外部es存储磁盘空间
