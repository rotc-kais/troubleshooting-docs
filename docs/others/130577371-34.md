# 3.4

证书被还原为1年有效期

## Cause
- oidc合法域名证书即将过期触发平台原证书更新策略

## Resolution
- 登录所有master节点执行：systemctl stop check-certs
- 登录所有master节点执行：systemctl disable check-certs
