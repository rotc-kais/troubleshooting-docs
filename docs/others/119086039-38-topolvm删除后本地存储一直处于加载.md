# 3.8 topolvm删除后本地存储一直处于加载

本地存储一直处于加载中 接口报错topolvmcluster CRD不存在

## Cause
- 3.8版本存在已知bug

## Resolution
- 升级到3.8.1版本修复该问题

## [workaround]
- 修改URL地址，将访问路径中的view替换为config
