# 本地存储添加存储节点界面状态显示未知

界面显示存储节点状态未知 集群状态告警 topolvm-prepare日志显示磁盘存在文件系统

## Cause
- 使用的磁盘存在文件系统
- 节点CPU资源不足

## Resolution
- 擦除磁盘分区并清除文件系统
- 补充节点CPU资源
