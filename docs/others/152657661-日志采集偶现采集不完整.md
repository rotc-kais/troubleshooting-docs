# 日志采集偶现采集不完整

日志采集偶现不完整 查询时行数缺失甚至无记录

## Cause
- 容器启动/退出触发nevermore日志采集进程重启
- 节点pod数量过多

## Resolution
- 确认该问题为TKE 3.10版本缺陷，需升级版本修复
