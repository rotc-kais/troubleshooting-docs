# 0

大量驱逐的pod

## Cause
- 集群存储使用达到上限

## Resolution
- 执行命令：kubectl get pod --all-namespaces |grep Evicted | awk '{print $1" "$2}'|while read line ; do kubectl delete pod -n $line ; done
