# 3.10

子网状态持续显示为'删除中'无法完成删除

## Cause
- 已分配的IP地址未被释放导致子网删除受阻

## Resolution
- 执行命令检查残留IP资源: kubectl get ips -A | grep 192.168.202.121
- 手动删除残留IP资源: kubectl delete ips xxxxxxxxxxxx
