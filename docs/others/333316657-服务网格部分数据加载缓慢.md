# 服务网格部分数据加载缓慢

部分数据加载缓慢 浏览器接口可能存在报错 存在143个请求堆积

## Cause
- Chrome浏览器对同域名存在5个并发限制
- F5负载均衡器未支持HTTP2协议

## Resolution
- 升级F5负载均衡器支持HTTP2协议

## [workaround]
- 在浏览器localstorage设置enabled_downgrade_watch=true缓解问题
