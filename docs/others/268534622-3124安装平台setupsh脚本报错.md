# 3.12.4安装平台setup.sh脚本报错

运行 bash setup.sh 时立即报错

## Cause
- /etc/resolv.conf 存在隐藏不可修改权限导致脚本检测失败

## Resolution
- 执行 chattr -i /etc/resolv.conf 移除隐藏权限
- 完成部署后恢复权限：chattr +i /etc/resolv.conf

## [workaround]
