# 3.4.2

应用部署数量显示不准确（显示19个，实际应有18个） 应用yaml中存在stop状态的deployment历史记录

## Cause
- 其他命名空间的应用使用了与此应用相同的label
- application的matchlabel配置匹配了全集群下所有同名label的deployment

## Resolution
- 删除其他命名空间下应用的相同label
