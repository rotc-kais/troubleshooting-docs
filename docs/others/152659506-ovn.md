# ovn

ovn-pinger异常无法启动 报错network：plugin type="multus" name="multus-cni-network' failed

## Cause
- 存在冲突的CNI配置（00-kube-ovn.conflist文件与multus.d目录）

## Resolution
- cd /etc/cni/net.d/
- 删除00-kube-ovn.conflist文件和multus.d目录
- 重建multus服务
