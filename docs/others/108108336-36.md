# 3.6

部署knative-operator后状态显示'未初始化' 卸载后重新部署问题依旧存在

## Cause
- 存在未删除的knative对应csv资源

## Resolution
- 删除业务集群的knative对应csv资源
- 通过页面重新部署knative-operator
