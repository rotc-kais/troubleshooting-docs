# 3.4.2

无法发布容器 执行命令报错'bash fork: cannot allocate memory' 主机进程数接近最大上限

## Cause
- 主机进程数达到内核限制(kernel.pid_max)

## Resolution
- echo "kernel.pid_max=1000000" >> /etc/sysctl.conf
- sysctl -p
