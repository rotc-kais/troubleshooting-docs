# ASM调用链无数据

调用链查询无数据

## Cause
- 索引类型为'text'而非'keyword'
- 暴力删除索引或删除Elasticsearch数据目录导致索引类型错误

## Resolution
- 清除对应的索引，重启jaeger-collector和jaeger-query组件生成新索引
