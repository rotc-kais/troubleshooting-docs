# GPU节点部署问题

cpaas-monitor-exporter-gpu pod启动异常 global集群获取gpu-manager版本信息为空 业务集群已开启虚拟GPU

## Cause
- 客户安装的CUDA版本(12.2)与gpu-manager支持的CUDA版本(11.4)不适配

## Resolution
- 重新安装nvidia驱动，更换适配版本为535.54.03（对应CUDA 11.4）
