# 创建集群添加节点

执行命令 `ip route show | grep 'default\|0.0.0.0'` 返回为空

## Cause
- `/etc/sysconfig/network-scripts/ifcfg-eth0` 未配置默认网关

## Resolution
- 联系 IaaS 层负责人配置默认网关后重新操作
