# 容器exec问题

exec failed: unable to start container process: open /dev/pts/0: operation not permitted: unknow

## Cause
- runc v1.1.3 中不再添加 DeviceAllow=char-pts rwm 规则
- 执行 systemctl daemon-reload 导致规则丢失

## Resolution
- 重启容器
