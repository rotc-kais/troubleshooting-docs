# 通过多架构镜像文件进行tag命名后，新tag镜像无法使用

新tag镜像无法使用

## Cause
- ARM环境下更改tag导致架构/操作系统/依赖项不兼容
- 镜像损坏

## Resolution
- 使用支持多架构的构建工具重新构建镜像
- 确保tag操作保留原始manifest信息
