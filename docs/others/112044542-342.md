# 3.4.2

TKE页面访问平台管理-集群页面显示加载错误 重新加载后3秒左右报错

## Cause
- nginx配置中proxy_connect_timeout和proxy_timeout均为3s

## Resolution
- 调整nginx配置文件内proxy_connect_timeout和proxy_timeout配置为60s
