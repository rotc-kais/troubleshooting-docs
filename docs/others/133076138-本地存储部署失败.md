# 本地存储部署失败

本地存储部署失败 topolvm部署失败

## Cause
- 残留的旧环境配置/数据导致部署失败

## Resolution
- kubectl delete topolvmcluster
- 执行磁盘清理命令：dd if=/dev/zero of=/dev/sdX bs=1M count=100
- 重新部署topolvm
