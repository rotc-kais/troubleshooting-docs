# 水平伸缩增加自定义指标

水平伸缩策略基于错误的资源使用数据触发伸缩

## Cause
- 默认CPU/内存指标基于pod requests配置计算，未考虑集群超售比设置

## Resolution
- 创建原生应用并通过资源管理踢出原有资源
- 导入包含自定义指标的deployment
- 在应用监控中拷贝CPU/内存监控表达式
- 创建自定义监控面板并配置弹性伸缩策略
