# 4.0 部署aml失败

failed to install kserveless operator: context deadline exceeded kubectl get amlcluster default 显示状态为 False，原因为 Failed

## Cause
- 未部署服务网格导致缺少 asm.asm.operator.alauda.io/v1alpha1 CRD 资源

## Resolution
- 部署服务网格以补全缺失的 CRD 资源
