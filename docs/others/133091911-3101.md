# 3.10.1

nevermore容器健康检查失败 Prometheus采集Metrics的Service(nevermore)无法访问 /cpaas/log/log.err.log文件权限异常

## Cause
- 主机存储盘符变更导致文件权限变更

## Resolution
- 卸载日志采集组件
- 删除/cpaas目录下audit event log目录
- 重新部署日志采集组件
