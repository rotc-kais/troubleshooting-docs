# udp测试丢包问题

测试方法一中，包大小1500比5000丢包严重 测试方法二中，每次测试第一秒存在丢包，中间过程偶发丢包

## Cause
- UDP接收缓冲区（rmem_default）设置不足导致丢包

## Resolution
- 调整内核参数：将rmem_default设置为2097152
