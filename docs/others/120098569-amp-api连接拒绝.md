# amp api连接拒绝

30000端口连接拒绝 前端无法修改API地址

## Cause
- amp的svc配置externalTrafficPolicy=local导致nodeport无法通过平台vip访问

## Resolution
- 将externalTrafficPolicy更新为cluster
