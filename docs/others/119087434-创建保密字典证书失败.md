# 创建保密字典证书失败

创建保密字典证书失败 报错不识别私钥类型 不识别BEGIN RSA PRIVATE KEY

## Cause
- 证书私钥存在密码保护
- 平台使用openssl解密时无法识别带密码的RSA格式私钥

## Resolution
- 使用命令去除私钥密码保护：openssl rsa -in encrypted.key -out decrypted.key
