# GPU使用问题

容器内运行nvidia-smi pmon -s u -d 1无数据 GPU使用率监控显示83%但limit为50（1个物理核心的50%） 显存监控显示6G但limit为4u（1G）

## Cause
- Ubuntu 2204使用cgroup v2导致gpu-manager不兼容

## Resolution
- 使用hami的vgpu方案替代gpu-manager
