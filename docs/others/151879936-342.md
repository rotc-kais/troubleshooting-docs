# 3.4.2

exporter-kube-state oom

## Cause
- 资源分配不足导致内存溢出

## Resolution
- kubectl -n cpaas-system edit deploy kube-prometheus-exporter-kube-state
调大参数：
- --extra-memory=200Mi
- --extra-cpu=50m

## [workaround]
