# 更新域名数据开启证书的公钥私钥没有校验

更新域名开启证书时，不填写公钥和私钥也能成功更新

## Cause
- 平台未对域名证书更新时的公钥和私钥字段做非空校验

## Resolution
- 等待后续版本优化解决证书校验逻辑
