# 入站网关添加https协议监听时，无法选择证书

添加https协议监听时无法选择证书（域名已配置证书）

## Cause
- 证书未颁发给对应域名
- 证书缺少dnsname信息

## Resolution
- 重新签发符合要求的证书并更新
