# 3.8.2

部署topolvm显示设备类不可用

## Cause
- 设备类名称与磁盘盘符名称相同导致识别冲突（例如设备类名称vdb与磁盘路径/dev/vdb）

## Resolution
- 修改设备类名称(如hdd/ssd)后重新部署
