# 平台的反向代理支持 Websocket

页面显示：远程中断连接已断开，请重新连接

## Cause
- nginx的http反向代理缺少Upgrade相关配置

## Resolution
- 改为TCP模式反向代理（nginx/haproxy）
- 在nginx配置中添加：proxy_set_header Upgrade $http_upgrade; proxy_set_header Connection upgrade; 并重启服务
