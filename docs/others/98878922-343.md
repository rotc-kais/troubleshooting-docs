# 3.4.3

部署提示port connection check不通过 /usr/local/sbin/ncat: Permission denied

## Cause
- 3.4.3版本基线x86版本不支持麒麟操作系统
- ncat命令权限异常(600)

## Resolution
- 修改ncat权限为755: chmod 755 /usr/local/sbin/ncat
