# 3.12.2ceph

数据均衡期间可能触发存储告警状态

## Cause
- 新增裸盘后触发数据重平衡过程

## Resolution
- 等待数据自动均衡完成

## [workaround]
- 如需回退需移除新增磁盘并再次完成数据均衡
