# 3.6.1

健康检查经常性不通过 pod不断重启

## Cause
- metrics代码问题导致客户将主机的自定义规则都挂到了一个deployment下

## Resolution
- 确定为bug，产研排期解决

## [workaround]
- 避免将主机的自定义规则挂载到同一deployment下
