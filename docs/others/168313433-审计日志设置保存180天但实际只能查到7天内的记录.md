# 审计日志设置保存180天，但实际只能查到7天内的记录

审计日志设置保留180天，实际只能查询7天内的记录

## Cause
- moduleinfo日志存储策略中global logcenter的event保留时间被设置为7天

## Resolution
- 将moduleinfo日志存储策略中event的ttl值从7天调整为180天
