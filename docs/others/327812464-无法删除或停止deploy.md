# 无法删除或停止deploy

无法删除或停止Deployment 修改操作被自动还原 节点选择器标签对应的节点已移出集群

## Cause
- 存在Operator持续同步配置
- 节点标签变更导致调度策略失效

## Resolution
- 在Deployment的metadata.annotations添加skip-sync: "true"
- 修改对应Operator实例的配置

## [workaround]
- 通过annotations临时禁用同步：metadata.annotations: skip-sync: "true"
