# 创建业务集群节点检测不过

节点检测失败报tmp EOF错误 端口检测不通过

## Cause
- 业务集群节点sftp服务未开启

## Resolution
- 将正常节点的/usr/libexec/openssh/sftp-server复制到异常节点
- 开启sftp服务
