# 添加节点界面显示异常

添加节点界面显示异常 global集群machine资源显示节点not found

## Cause
- 首次删除节点时未执行cleanup脚本直接重新加入集群

## Resolution
- 在global集群执行kubectl delete mc [异常的mc资源]
- 在节点执行cleanup脚本后重新加入集群
