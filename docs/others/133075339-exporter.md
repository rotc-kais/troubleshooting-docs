# exporter

metrics请求返回很慢 exporter-node日志报错: write tcp ...: write: broken pipe

## Cause
- Ubuntu系统下exporter-node组件可能因GOMAXPROCS设置不当导致资源争用

## Resolution
- 给kube-prometheus-exporter-node增加环境变量: name: GOMAXPROCS, value: "1"

## [workaround]
- 同resolution方案
