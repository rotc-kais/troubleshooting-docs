# 项目管理页面加载失败

项目管理页面加载失败 切换页面后项目不可见 错误信息: no matches for kind "Project" in version "auth.alauda.io/v1"

## Cause
- 部署顺序问题导致apollo组件在CRD注册前启动

## Resolution
- 重启apollo组件使其重新检测已注册的CRD
