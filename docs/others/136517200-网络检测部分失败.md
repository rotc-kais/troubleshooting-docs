# 网络检测部分失败

网络检测部分失败 curl不带-I参数时通讯失败 系统日志存在mtu不一致报错

## Cause
- 物理网卡mtu被从1400修改为1500
- ovn0网卡mtu(1300)与pod mtu(1400)不一致

## Resolution
- 将物理网卡mtu从1500改回1400
- 重启所有被修改过的kube-ovn-cni pod
