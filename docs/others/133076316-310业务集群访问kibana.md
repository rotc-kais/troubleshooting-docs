# 3.10业务集群访问kibana

## 环境信息 适用版本：3.10

## Cause

## Resolution
- kubectl get feature -n xxxxx cpaas-elasticsearch -oyaml 获取elasticsearchUrl地址，将es_proxy替换为kibana
- kubectl get secret elasticsearch-basic-auth 获取密码并base64解码
