# 接入集群部署监控失败

no matches for kind "certficate" in version "cert-manager.io/v1beta1" ars部署失败

## Cause
- certificates.cert-manager.io 这个 CRD 被修改过

## Resolution
- 使用正常集群的 certificates.crd 覆盖问题集群的 CRD
