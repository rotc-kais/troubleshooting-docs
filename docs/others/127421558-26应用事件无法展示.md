# 2.6应用事件无法展示

应用事件无法展示 管理视图-事件仅能加载3小时前的事件 ES集群及event相关索引处于red状态

## Cause
- ES集群处于red状态导致事件索引不可用

## Resolution
- 横向增加ES节点
- 纵向扩容ES节点CPU/内存资源
- 更换高性能磁盘存储
- 调整索引分配数量
