# 3.*

添加节点时报错私钥认证失败

## Cause
- 使用了业务集群master节点到新节点的私钥，而非global master节点到新节点的密钥对

## Resolution
- 重新生成global master节点到新节点的SSH密钥对，并重新添加节点
