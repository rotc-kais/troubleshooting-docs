# 3.4.2

集群过多导致ConfigMap数量过大

## Cause
- 默认保留20次巡检记录

## Resolution
- 设置inspection cr的保存数量参数: --inspection-retention-number=20
- 修改aiops chart的courier.arguments配置，添加- --inspection-retention-number=20

## [workaround]
