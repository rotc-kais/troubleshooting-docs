# 3.0.4通知服务器设置SSL方式后，无法正常添加邮件发送人

无法正常添加邮件发送人 报错500

## Cause
- 创建通知服务器时未正确配置SSL参数

## Resolution
- 在NotificationServer资源的.spec字段添加sslEnabled: true配置项
