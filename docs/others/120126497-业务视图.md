# 业务视图

业务视图-部署界面无法加载实例详细信息 F12存在401报错 部分节点业务偶现加载失败

## Cause
- 业务集群部分节点无法解析平台域名
- kube-apiservice服务请求平台域名失败

## Resolution
- 在问题节点/etc/hosts添加平台域名解析
- 重启kube-apiservice容器服务
