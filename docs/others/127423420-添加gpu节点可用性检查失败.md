# 添加gpu节点可用性检查失败

添加GPU节点时节点可用性检查报错：dont has GPU hard

## Cause
- 节点缺少lspci命令导致无法获取GPU卡信息

## Resolution
- yum install -y pciutils 安装lspci命令
