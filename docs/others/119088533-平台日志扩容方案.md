# 平台日志扩容方案

JIRA Jira：

## Cause

## Resolution
- 修改global集群prdb base资源配置：kubectl edit prdb base
- 在.spec.valuesoverride添加配置：
  valuesOverride:
    ait/chart-elasticsearch:
      cpaasElasticsearch:
        resources:
          limits:
            cpu: "4"
            memory: 8Gi
          requests:
            cpu: "2"
            memory: 4Gi
      elasticsearch:
        java_opts: -Xms6g -Xmx6g
- 检查配置生效：kubectl get deployment -n cpaas-system cpaas-elasticsearch -oyaml
