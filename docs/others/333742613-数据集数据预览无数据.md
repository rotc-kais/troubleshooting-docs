# 数据集数据预览无数据

数据集数据预览无数据 amlservice日志报错Table 'aml.datasets_meta' doesn't exist

## Cause
- 数据库存在但缺少表结构
- 代码逻辑在数据库存在时跳过表同步操作

## Resolution
- 手动删除aml数据库
- 触发缓存重建流程自动创建数据库和表结构
