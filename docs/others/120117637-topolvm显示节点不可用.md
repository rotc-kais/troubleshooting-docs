# topolvm显示节点不可用

部署到节点vdb、vdc时异常 卸载后重新部署问题依然存在

## Cause
- 未安装lvm2 yum包导致缺少相关命令

## Resolution
- 在不可用节点执行命令：yum install lvm2 -y
