# 内部路由通过标签选择，无法匹配上已打上标签的后端

无法匹配上已打上标签的后端Pod

## Cause
- Service配置使用了Deployment层级的标签而非Pod模板(template)层级的标签

## Resolution
- 检查并修改Service的selector配置，使其匹配Deployment中spec.template.metadata.labels定义的标签
