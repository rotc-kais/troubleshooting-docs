# 创建业务集群节点检查不通过

ncat安装失败 集群节点6443端口无法访问 手动拷贝速度较慢

## Cause
- 云上global集群与云下业务集群间网络带宽仅5Mbps
- 不符合部署要求的百兆带宽标准

## Resolution
- 将网络带宽升级至百兆以上
