# 平台部署

业务集群部署失败 prometheus-adapter 组件持续崩溃（CrashLoopBackOff） init apprelease 部署停滞

## Cause
- monitor 组件过早部署
- 依赖组件 cluster-base 未完成部署

## Resolution
- 创建业务集群时不勾选监控部署选项
- 集群安装成功后通过独立流程部署监控组件
