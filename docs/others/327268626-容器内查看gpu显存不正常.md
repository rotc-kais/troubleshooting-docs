# 容器内查看gpu显存不正常

容器内执行nvidia-smi显存显示异常 程序报段错误

## Cause
- nvidia-driver版本(12.2)与gpu-manager 3.12.2支持的CUDA版本(11.6)不兼容

## Resolution
- 降低节点的CUDA版本
- 改用整卡模式运行
