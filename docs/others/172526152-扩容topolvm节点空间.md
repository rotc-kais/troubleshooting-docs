# 扩容topolvm节点空间

## 环境信息 适用版本：3.12.1

## Cause

## Resolution
- fdisk查看/dev/sdb已分区100G，还剩100G
- 添加新分区/dev/sdb3，容量为100G
- fdisk确认/dev/sdb已经有新分区
- 通过平台管理界面在本地存储添加存储设备
- 选择对应节点、相同设备类和新加存储设备
- 等待10分钟后刷新页面，等待topolvm后台扩容完成
