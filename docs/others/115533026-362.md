# 3.6.2

创建项目时无法配置GPU配额选项 业务创建时无GPU配额选项

## Cause
- 缺少cf-crl-nvidia-device-plugin configmap资源

## Resolution
- 创建cf-crl-nvidia-device-plugin configmap资源并重启GPU相关pod
