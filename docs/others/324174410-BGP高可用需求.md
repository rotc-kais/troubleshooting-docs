# BGP高可用需求

需要在同一集群添加多个BGP对等体peer地址，实现到核心路由器的高可用选路

## Cause
- 配置多个BGP peer时，每个节点上的speaker会与每个peer建立BGP连接，形成peer数 × 节点数的BGP会话，通过ECMP路径实现高可用选路

## Resolution
- 配置多个BGP peer，每个peer对应核心路由器的地址，并确保正确配置
