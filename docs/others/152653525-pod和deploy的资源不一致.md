# pod和deploy的资源不一致

pod的资源声明（2c4g）与deployment的request.cpu:100m、memory:100m不一致

## Cause
- metis组件可能覆盖了资源配比设置

## Resolution
- 修改kubectl get features resourceratio配置项

## [workaround]
- 关闭metis的pod后执行delete pod操作
