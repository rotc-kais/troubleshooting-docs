# hami+910部署qwen2

模型启动失败 错误日志显示底层设备识别异常 无法识别底层 NPU 信息

## Cause
- 底层 NPU 设备识别异常
- 需要 NPU 提供侧进行硬件/驱动层排查

## Resolution
- 联系底层 NPU 支持团队进行设备识别异常排查
- 检查 NPU 驱动版本与模型要求的兼容性
