# 灵雀云3.16使用GPU

模型初始化失败，cuda error invalid device context

## Cause
- CUDA 运行时版本(11.3)与节点驱动版本(11.6)不兼容导致 cumemalloc 函数异常

## Resolution
- 替换为 hami 组件（在 CUDA 10.1/11.3 运行时验证通过）
