# 测试环境几个集群同时出现underlay网络不通

underlay网络访问api-server偶发不通 出现不通的频率较高

## Cause
- 华三交换机的ND表项已满
- ND表项过多占用nexthoppool1资源
- 路由无法正常配置

## Resolution
- 修改ND老化时间从240分钟调整为20分钟
