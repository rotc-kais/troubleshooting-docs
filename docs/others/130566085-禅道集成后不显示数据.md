# 禅道集成后不显示数据

禅道集成后不显示数据 katanomi plugin pod日志显示处理zentao请求反序列化失败

## Cause
- zentao使用的版本是非TKE兼容版本

## Resolution
- 更换为平台支持的禅道版本
