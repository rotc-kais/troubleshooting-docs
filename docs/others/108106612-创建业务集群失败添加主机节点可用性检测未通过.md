# 创建业务集群失败，添加主机节点可用性检测未通过

节点可用性检测失败无法添加 sftp连接异常

## Cause
- 缺少sftp-server执行文件

## Resolution
- 将正常节点的sftp-server文件拷贝至故障节点相同路径
