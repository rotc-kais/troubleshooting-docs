# es磁盘空间满了之后kibana访问异常

kibana访问404 es磁盘空间满

## Cause
- es磁盘满导致集群异常，影响kibana数据访问

## Resolution
- 通过curl命令删除es索引：`curl -XDELETE -u admin:$PASSWORD "http://$SVC:9200/$indices"`
