# katanomi

katanomi-init pod 发生了 OOM（内存不足）错误，导致容器被终止

## Cause
- 内存限制配置不合理
- 存在内存泄漏或异常高内存消耗

## Resolution
- 调整 katanomi-init pod 的内存限制
- 优化容器的内存使用
- 监控调整后的内存使用情况
