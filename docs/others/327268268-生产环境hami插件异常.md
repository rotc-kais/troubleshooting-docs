# 生产环境hami插件异常

无法部署GPU服务

## Cause
- 新增GPU节点后未初始化
- 未将default runtime设置为nvidia

## Resolution
- 初始化新增GPU节点
- 在docker配置中设置"default-runtime": "nvidia"
