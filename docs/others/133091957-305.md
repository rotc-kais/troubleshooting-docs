# 3.0.5

sentry连接数很多 tcp断开连接时长30s

## Cause
- sentry组件每次请求新建httpClient，默认keepalive方式导致长时间不释放

## Resolution
- 3.6.3版本修复
- 临时重启sentry组件恢复
