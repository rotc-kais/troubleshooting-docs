# GPU 底层插件排查逻辑文档

cpaas-monitor-exporter-gpu pod启动异常 global集群获取gpu-manager版本信息为空

## Cause
- CUDA 12.2版本与gpu-manager支持的CUDA 11.4版本不兼容

## Resolution
- 重新安装nvidia驱动，更换适配版本（推荐版本见原始文档图片）
