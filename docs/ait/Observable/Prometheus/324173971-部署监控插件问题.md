# 部署监控插件问题

grafana访问失败 自定义grafana面板被删除 实时告警提示prometheus pod状态异常

## Cause
- 未备份grafana目录数据导致访问失败
- 旧prometheus-thanos/prometheus-operator pod未正常终止

## Resolution
- 部署前备份1、2节点grafana目录数据
- 手动删除prometheus-thanos和prometheus-operator异常pod
