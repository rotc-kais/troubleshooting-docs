# 业务集群监控组件部署异常

页面不显示监控数据且一直提示部署中 global集群检查发现prometheus-0的sts未找到 业务集群缺少prometheus-0容器和sts

## Cause
- 存在表达式为空的prometheusrules触发平台bug，导致sts无法创建

## Resolution
- 清理所有表达式为空的prometheusrules
