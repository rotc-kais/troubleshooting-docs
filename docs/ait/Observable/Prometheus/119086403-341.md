# 3.4.1

业务集群部署prometheus后台没有容器启动 部署新监控时卡住

## Cause
- 节点122存在未清理的gd监控残留资源
- 资源finalizer字段阻断删除操作

## Resolution
- 手动清理现有监控残留资源
- 移除资源finalizer字段后强制删除
- 删除残留pod资源后重新部署监控
