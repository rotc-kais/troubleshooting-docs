# 3.4.5 exporter

容器状态为crashbackoff 日志报错no such file or directory 应用监控无法查看

## Cause
- 修改漏洞时替换了base镜像导致二进制缺少glibc依赖

## Resolution
- 更新该组件镜像版本
- 升级平台版本
