# 集群告警策略对应哪个资源，是否可以通过kubectl进行创建

JIRA Jira：

## Cause
- 告警策略由Prometheus Operator管理的PrometheusRule CRD实现

## Resolution
- 使用kubectl get/apply/delete prometheusrules进行管理
- 创建符合PrometheusRule规范的YAML文件
