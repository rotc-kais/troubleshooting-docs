# 3.10.2对接cce集群后不显示监控数据

不显示监控数据 界面报错“监控组件异常，只能获取部分监控数据，请修复”

## Cause
- 业务集群外网地址:11780网络不通

## Resolution
- 放开防火墙对11780端口的访问权限
