# underlay 集群，节点监听到 非该节点上的通信报文

节点上抓到非本节点Pod的通信报文 ALB Pods运行但无端口监听

## Cause
- vswitch开启了混杂模式

## Resolution
- 检查并关闭vswitch的混杂模式
