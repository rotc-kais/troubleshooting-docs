# global监控无法展示，报错502

global监控无法展示，报错502 alb2访问记录显示no balancer found for global-xxx

## Cause
- 3.6升级3.8后残留cluster-proxy-global ingress规则
- ingress规则中svc端口与实际svc端口不一致

## Resolution
- 删除残留的cluster-proxy-global规则
