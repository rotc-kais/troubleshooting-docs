# 日志

日志加载延迟无法查看到近15分钟日志，需要调整到较大时间才能进行日志加载

## Cause
- kafka数据堆积
- es节点磁盘写入阻塞
- 磁盘iops或吞吐量不足
- es节点cpu或内存资源瓶颈

## Resolution
- 更换高性能ssd存储
- 根据部署文档调整日志组件配置

## [workaround]
- 调整查询时间范围到较大区间
