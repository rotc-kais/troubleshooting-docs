# 3.8.2

监控面板502错误 warlock组件日志显示用户密码错误

## Cause
- kube-prometheus-grafana-busic-auth secret中的密码不正确

## Resolution
- 修改kube-prometheus-grafana-busic-auth secret的password字段为admin
- 重启warlock pod

## [workaround]
