# 重新部署监控后grafana面板丢失

监控面板点开后显示无面板

## Cause
- grafanadashboard资源残留
- sentry同步失败

## Resolution
- 删除所有grafanadashboard资源
- 删除所有和grafanadashboard相关的rpch
- 重启sentry容器
- 手动更新monitor的ars(在values下添加任意kv键值触发同步)
