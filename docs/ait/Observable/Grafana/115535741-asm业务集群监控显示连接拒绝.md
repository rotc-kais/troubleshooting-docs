# asm业务集群监控显示连接拒绝

监控无法显示，报错拒绝连接 请求头报错 grafana环境变量协议不一致

## Cause
- grafana配置的访问地址为https协议，平台使用http协议访问

## Resolution
- kubectl edit ClusterConfig asm-cluster-config 修改grafanaURL
