# 网格列表中的监控异常

网格列表中的监控异常 grafana服务pod存在重启现象

## Cause
- 服务oom导致重启
- grafana所在节点资源使用率过高

## Resolution
- kubectl edit deployment -n <namespace> 增加grafana服务的requests资源配置，使其与limits保持一致
