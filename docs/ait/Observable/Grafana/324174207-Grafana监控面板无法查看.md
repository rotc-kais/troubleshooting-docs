# Grafana监控面板无法查看

返回状态码500 Grafana对应Pod状态异常，容器组重启无法成功 日志报错no space

## Cause
- 节点/cpaas目录存储空间耗尽
- 集群es卸载后残留日志数据未清理

## Resolution
- 清理/cpaas目录下的日志数据
- 删除已卸载es集群的残留日志数据
