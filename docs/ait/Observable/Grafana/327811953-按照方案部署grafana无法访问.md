# 按照方案部署grafana无法访问

访问Grafana返回404错误 NodePort访问跳转路径后显示找不到网页

## Cause
- grafana-operator与grafana实例使用的社区镜像存在不兼容问题

## Resolution
- 更换为方案中指定的grafana实例镜像
