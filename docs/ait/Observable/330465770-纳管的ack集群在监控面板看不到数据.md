# 纳管的ack集群，在监控面板看不到数据

监控面板看不到数据 未触发告警

## Cause
- 容器网络到vm存储集群vip:11780端口不通

## Resolution
- 开通业务集群监控组件容器网络到vm存储集群vip:11780端口转发
