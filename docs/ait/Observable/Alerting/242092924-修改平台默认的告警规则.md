# 修改平台默认的告警规则

无法在界面直接修改告警策略的表达式，只能修改告警触发条件

## Cause

## Resolution
- 在对应集群的master节点执行命令 kubectl edit alertrules.ait.alauda.io -n cpaas-system cpaas-node-rules 修改YAML配置
