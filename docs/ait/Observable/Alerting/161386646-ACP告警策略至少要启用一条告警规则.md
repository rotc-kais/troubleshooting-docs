# ACP告警策略至少要启用一条告警规则

创建告警策略时必须至少启用一条告警规则 删除最后一条告警规则时报错

## Cause
- 平台逻辑强制要求告警策略必须包含至少一条生效的告警规则

## Resolution
- 删除不含告警规则的策略
- 对策略进行静默处理
