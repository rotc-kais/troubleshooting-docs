# 告警规则优化

磁盘空间使用率监控对根目录(/)无效，仅监控到/var/lib/containerd路径

## Cause
- 3.18.1版本存在监控规则配置缺陷
- 监控规则仅配置了containerd存储目录的磁盘检测

## Resolution
- 升级到4.0版本获得修复
