# 告警策略那里添加之后资源对象就置灰了，不能调整

告警策略添加后资源对象置灰，无法调整或删除资源对象

## Cause
- 当前版本设计限制导致资源对象与告警策略绑定后不可编辑

## Resolution
- 等待未来版本功能优化

## [workaround]
- 需先删除关联的告警策略才能重新调整资源对象
