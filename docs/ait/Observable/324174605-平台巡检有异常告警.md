# 平台巡检有异常告警

registry pod异常卡在init chi-pxc-ck pod异常 discover-device pod因OOM异常终止

## Cause
- 退出的容器占用了端口
- PVC存储已满
- 内存资源限制过低（默认128m）
- 磁盘故障导致pvs和fsType异常

## Resolution
- 清理占用端口的已退出容器
- 扩容PVC存储容量
- 调整内存资源限制至256m
- 将故障磁盘从集群中移除并重新添加
