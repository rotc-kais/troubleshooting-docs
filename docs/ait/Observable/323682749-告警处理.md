# 告警处理

olm-registry容器组CPU使用率周期性上涨 istio证书有效期小于30天告警

## Cause
- OOM导致CPU使用率上涨
- istio证书15天自动续期机制触发有效期告警

## Resolution
- 增加olm-registry资源配置
- 修改olm-registry-config configmap后重启artifact-controller
- 忽略istio证书有效期告警
