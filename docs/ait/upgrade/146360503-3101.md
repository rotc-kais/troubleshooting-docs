# 3.10.1

健康探测audit目录不存在 频繁触发组件重启

## Cause
- 节点内缺少audit.log结尾的日志文件
- nevermore组件存在未勾选的日志类型配置

## Resolution
- 临时方案：在配置中勾选全部日志类型
- 永久方案：升级平台版本至3.12
