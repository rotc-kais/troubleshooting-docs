# global集群基础功能组件一直处于处理中状态

global集群基础功能组件一直处于处理中状态 ars状态异常(syncing) sentry副本数为0

## Cause
- 产品设计缺陷导致sentry deployment变更时被记录到rpch
- rpch记录阻止sentry组件正常拉起

## Resolution
- 删除记录sentry副本的rpch资源
