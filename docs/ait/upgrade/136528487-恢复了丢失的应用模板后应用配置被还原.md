# 恢复了丢失的应用模板后，应用配置被还原

升级后本地模板仓库中的模板丢失 手动恢复模板后应用配置被还原

## Cause
- captain-chartmuseum调度变化导致模板丢失
- 平台升级/应用异常触发captain重新同步模板

## Resolution
- 通过value配置模板应用保证后续同步依据
- 将应用与模板脱离转为原生应用
