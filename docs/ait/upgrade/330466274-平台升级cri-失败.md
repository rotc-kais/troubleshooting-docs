# 平台升级cri 失败

平台升级containerd cri 失败。报错信息如图 原因是由于 /usr 挂载到了主机中，大多数os上有 /usr/bin/sh，sh 真正将使用 /usr/bin/sh，不会有问题。但Astra OS 的/usr/bin/sh 指向了 /bin/bash，而在容器中不存在 /bin/bash，报错。
