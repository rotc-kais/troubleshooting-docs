# 数广财厅flannel 从3.8 升级 到 3.12 端口变化导致跨节点不通

升级后所有跨节点pod流量都不通

## Cause
- flannel端口从8472变更为4789导致防火墙拦截

## Resolution
- 修改kube-flannel-cfg configmap的端口配置为8472
