# 集群节点监控信息断续

集群节点监控信息断续

## Cause
- 历史遗留问题（一直存在的bug）

## Resolution
- 升级平台版本至3.12版本解决

## [workaround]
- 临时重启对应集群kube-prometheus-exporter-kube-state服务
