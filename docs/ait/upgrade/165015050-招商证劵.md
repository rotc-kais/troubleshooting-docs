# 招商证劵

ovs-ovn周期性的cpu使用率增高 ovs-vswitchd.log中频繁出现ovs interface增删记录（1分钟100+次）

## Cause
- ovs接口频繁增删导致cpu升高
- 异常pod重启引发ovs操作

## Resolution
- 检查并处理异常重启的pod
