# 3.8.1

创建本地存储时报错 跳转至部署topolvm operator页面后部署失败 topolvm的Subscription未被生成

## Cause
- CSV资源可能处于异常状态导致olm无法正确创建Subscription

## Resolution
- kubectl delete csv -n nativestore-system topolvm-operator-xxxxx

## [workaround]
- 强制删除CSV触发olm重新创建资源
