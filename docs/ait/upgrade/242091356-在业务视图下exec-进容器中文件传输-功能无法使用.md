# 在业务视图下，exec 进容器中'文件传输' 功能无法使用

在业务视图下exec进入容器时'上传文件'功能无法使用

## Cause
- 平台版本<3.16且容器内tar --help输出为中文导致识别失败

## Resolution
- 设置容器环境变量 LANG=en_US.UTF-8 和 LC_ALL=en_US.UTF-8
- 升级平台至3.16及以上版本

## [workaround]
- 临时设置容器环境变量 LANG=en_US.UTF-8 和 LC_ALL=en_US.UTF-8
