# 升级到4.0.2后点击工具链管理，跳转到空白页

升级到4.0.2后点击工具链管理跳转到空白页 devops模块消失

## Cause
- devops相关monitordashboard旧版本未被删除导致更新卡住

## Resolution
- 手动在监控面板处删除devops相关面板
