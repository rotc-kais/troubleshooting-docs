# 3.14升级到3.18后，service mesh调用链没数据

service mesh调用链没数据 es里面没有索引数据 jaeger-prod-collector日志显示模版格式问题

## Cause
- Elasticsearch中存在有问题的索引模版

## Resolution
- 删除ES中与asm相关的有问题的索引模版
- 重启jaeger组件
