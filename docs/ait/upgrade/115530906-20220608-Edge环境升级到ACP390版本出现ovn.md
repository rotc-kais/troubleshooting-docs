# 20220608 Edge环境，升级到ACP3.9.0版本，出现ovn

ovn-central pod处于0/1 Running状态 大量应用Pod启动失败 svc endpoints无有效数据

## Cause
- ovn-central readiness检查脚本在添加label阶段耗时过长导致状态更新延迟

## Resolution
- 重建ovn-central和ovs-ovn的pod
