# 制品晋级执行后一直处于运行状态

制品晋级执行后一直处于运行中

## Cause
- 制品晋级目标名称空间与ASM网格所在名称空间相同，导致sidecar注入冲突

## Resolution
- 避免ASM网格所在名称空间与制品晋级目标名称空间处于同一个
