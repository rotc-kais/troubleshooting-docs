# 配置了通知webhook，接口方反映没收到

webhook通知失败，接口方未收到通知 courier日志中无相关webhook调用记录 配置的webhook地址为http协议但实际请求使用https协议

## Cause
- 3.16.1版本存在bug：webhook地址配置http协议时会被错误拼接为https协议

## Resolution
- 升级到3.16.2版本获得修复
- 将webhook地址改为https协议格式

## [workaround]
- 临时使用https协议的webhook地址进行配置
