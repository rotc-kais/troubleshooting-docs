# 3.0.5

captain组件一直卡在init初始化 组件事件显示找不到webhook资源 mutatingwebhookconfigurations和validatingwebhookconfigurations中无captain对应的webhook

## Cause
- secret内证书更新后未正确重建webhook资源

## Resolution
- 备份并删除captain组件的deploy资源
- 修改platform ars资源添加变量触发更新
