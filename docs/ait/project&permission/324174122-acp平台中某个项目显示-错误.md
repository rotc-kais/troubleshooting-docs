# acp平台中某个项目显示 错误

项目显示错误-113920 no matches for kind 'projectquota' in version 'auth.alauda.io/v1' 命名空间pods配额被更新后自动恢复默认值

## Cause
- auth-controller2跨集群client未watch到目标集群的projectquota资源

## Resolution
- 重启global环境的auth-controller2组件
