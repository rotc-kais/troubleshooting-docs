# 多项目管理角色创建

JIRA Jira：

## Cause

## Resolution
- 创建无权限平台角色，在 global 资源管理拷贝系统项目管理员规则
- 移除 Container Platform 下的命名空间资源管理权限
- 分配角色并验证权限
