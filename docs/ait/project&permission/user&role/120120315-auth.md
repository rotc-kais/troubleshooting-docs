# auth

auth-controller2 找不存在的集群报错

## Cause
- 已删除集群存在残留的命名空间管理员权限绑定

## Resolution
- 通过命令 kubectl get userbinding -l 'cpaas.io/cluster=<已删除集群名称>,auth.cpaas.io/role.level=namespace' 查询并删除对应 userbinding 资源
