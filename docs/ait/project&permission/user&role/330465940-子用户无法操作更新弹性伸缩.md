# 子用户无法操作更新弹性伸缩

子用户无法操作更新弹性伸缩

## Cause
- 子用户未勾选自动扩缩容管理的删除权限

## Resolution
- 在角色权限中为子用户勾选自动扩缩容管理的删除权限
