# 3.6.2

调用node_exporter服务的9100/metrics接口需要认证账号密码

## Cause
- TKE的node_exporter容器增加了证书校验机制

## Resolution
- 使用默认认证凭证：用户名kube-prometheus-exporter-node，密码kube-prometheus-exporter-node
