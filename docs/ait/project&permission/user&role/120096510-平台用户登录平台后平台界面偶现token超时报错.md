# 平台用户登录平台后平台界面偶现token超时报错

平台用户登录后界面偶现token超时报错

## Cause
- 多人共用账号导致redirectUri不一致
- 不同访问协议（如HTTP/HTTPS混合访问）引发问题

## Resolution
- 升级到3.9版本修复

## [workaround]
- 避免共用账号
- 仅开放HTTPS协议访问
