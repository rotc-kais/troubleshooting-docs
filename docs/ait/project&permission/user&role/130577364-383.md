# 3.8.3

Ensurecopyfiles部分报错

## Cause
- tke-platform-controller的extra-res带入了root权限的文件，导致nonroot用户无权限执行

## Resolution
- 修改tke-platform-controller的Deployment，在initContainers中添加securityContext: runAsGroup:0, runAsUser:0

## [workaround]
