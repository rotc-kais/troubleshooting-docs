# 平台历史告警保存多久，存储位置在哪里

JIRA Jira：

## Cause

## Resolution
- 平台历史告警默认保存30天，超过30天的告警数据会被自动清理
- 告警数据存储在平台的持久化存储中，具体位置可通过告警系统的配置查看
- 通过配置告警系统的存储策略可延长保存时间
