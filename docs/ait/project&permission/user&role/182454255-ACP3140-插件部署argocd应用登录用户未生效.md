# ACP3.14.0 插件部署argocd应用登录用户未生效

登录用户后返回应用首页，登录未生效

## Cause
- argocd CR缺少extraurl和.server.insecure配置
- 内部路由后端服务未正确选择80端口
- 入站规则未设置HTTP协议

## Resolution
- 在argocd CR添加extraurl参数指定访问地址，并配置.server.insecure
- 调整内部路由配置指向80端口
- 修改入站规则后端服务协议为HTTP
