# 项目下日志功能无法过滤应用

应用下拉只能看到'全部' 接口返回500错误 接口缺少start_time参数

## Cause
- 缺少start_time参数导致接口异常

## Resolution
- 修改Apollo容器参数：kubectl edit deploy -ncpaas-system apollo 搜索es-enablealias参数改为true
