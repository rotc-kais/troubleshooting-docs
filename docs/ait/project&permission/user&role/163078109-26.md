# 2.6

无法查看新建集群信息 请求返回403 业务集群clusterrolebinding缺少对应用户信息

## Cause
- 业务集群alauda-cluster-base hr的labelBaseDomain配置为cpaas.io导致不匹配

## Resolution
- 卸载业务集群alauda-cluster-base hr并重新部署（调整labelBaseDomain为alauda.io）
- 重启global集群auth-controller2组件

## [workaround]
