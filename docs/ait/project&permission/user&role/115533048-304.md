# 3.0.4

平台审计显示admin用户一直在删除相关容器组 cpaas-monitoring-grafana容器组被频繁删除

## Cause
- cluster-transformer组件存在设计缺陷导致持续删除cpaas-monitoring-grafana容器组

## Resolution
- 应用临时hotfix修复
- 升级平台版本到3.4及以上
