# 3.12.1

配置访问控制15分钟超时登出，实际5-10秒就登出

## Cause
- 平台用户安全策略中'关闭浏览器结束会话'配置与访问控制策略冲突

## Resolution
- 关闭用户安全策略的'关闭浏览器结束会话'配置
