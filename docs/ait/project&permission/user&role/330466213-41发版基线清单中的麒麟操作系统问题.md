# 4.1发版基线清单中的麒麟操作系统问题

nodeport流表丢失导致无法访问

## Cause
- 使用麒麟操作系统SP1版本

## Resolution
- 新部署环境必须升级到麒麟SP3版本

## [workaround]
- 老环境可升级ACP但需持续承担nodeport访问风险
