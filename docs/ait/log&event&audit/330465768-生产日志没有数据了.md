# 生产日志没有数据了

生产日志没有数据 kafka监控面板存在数据堆积 lanaya日志中存在ik分词器相关报错

## Cause
- 日志使用ik分词器存在部分日志无法插入
- 当前版本es组件存在bug

## Resolution
- 更换es组件hotfix镜像
- 重启lanaya服务
