# 灾备环境无法查询所有日志

灾备环境无法查询所有日志 运维中心页面查询日志返回11781端口拒绝连接 apollo日志显示业务集群11781端口拒绝连接

## Cause
- 业务集群HA未转发11781端口

## Resolution
- 配置11781端口转发
