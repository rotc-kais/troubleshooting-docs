# 应用日志查询，查看上文无响应

应用日志查询时查看上文无响应

## Cause
- 3.18.1版本日志查询需全量加载并聚合分析日志，日志量过大导致串行处理耗时

## Resolution
- 升级到4.0版本(已优化查询逻辑)
