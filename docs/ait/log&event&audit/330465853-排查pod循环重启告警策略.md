# 排查pod循环重启告警策略

pod循环重启 OOM错误 日志采集组件资源使用率高

## Cause
- 日志告警规则中的关键词匹配导致资源消耗过大（1w条日志匹配100条告警规则时产生高负载）

## Resolution
- 优化日志告警规则，减少冗余规则或改进匹配逻辑
