# 日志无法查询，计算组件无法显示

日志无法查询 计算组件无法显示 没有生成近期索引

## Cause
- Elasticsearch分片数量达到max_shards_per_node=1000限制

## Resolution
- 清理历史索引降低分片数量至1899

## [workaround]
