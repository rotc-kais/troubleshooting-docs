# 平台日志无法正常查看

平台日志无法正常查看 Elasticsearch集群状态异常 日志数据分片不可用

## Cause
- Elasticsearch分片处于不可用状态

## Resolution
- 执行Elasticsearch重新分片操作
- 删除不可恢复的分片
