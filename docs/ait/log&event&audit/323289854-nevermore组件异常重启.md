# nevermore组件异常重启

nevermore组件因连接API超时自动重启 node-problem-detector容器退出码137 执行/config/fdpresure-check.sh脚本失败

## Cause
- apiserver连接异常
- 文件描述符资源耗尽

## Resolution
- 检查节点到apiserver(10.0.4.1:443)网络连通性
- 使用curl -k https://10.0.4.1:443验证连接
- 检查并调整文件描述符限制
