# 3.14.0

创建pxc时报错502 erebus服务pod返回502错误 erebus容器存在重启行为

## Cause
- 3.14.0版本erebus服务触发空指针异常
- 3.14.0版本已知问题

## Resolution
- 替换erebus镜像为cloud-registry.alauda.cn/ait/erebus:v3.14.6
