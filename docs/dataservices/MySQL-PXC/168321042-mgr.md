# mgr

mgr-mysql实例提示连接数过大 当前连接数接近或达到200

## Cause
- 最大连接数参数设置过低（当前设置为200）

## Resolution
- 通过管理界面将max_connections参数调整为500，并手动重启实例使配置生效

## [workaround]
- 重启mysql实例以临时释放连接
