# MGR修改密码方案

实例启动失败

## Cause
- 修改密码前进行mysql pod缩减操作导致新老密码冲突

## Resolution
- 通过ALTER USER命令修改mysql root密码
- 更新{实例名称}-root-password保密字典的password字段
- 在容器平台重启MGR实例
