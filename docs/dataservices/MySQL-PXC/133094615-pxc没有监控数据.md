# pxc没有监控数据

PXC监控数据无法加载

## Cause
- 监控查询表达式配置错误

## Resolution
- 手动修改Grafana的dashboard配置：修改Variables->vmcluster的Type为Text box，Text Options设置为当前PXC所在集群名称
