# mysql

proxysql->pxc异常，单用户最大连接数超出上限

## Cause
- 单个数据库用户的连接数超过max_user_connections限制

## Resolution
- 通过平台参数修改mysql.user表中对应用户的max_user_connections值
- 重启mysql服务使配置生效
