# PXC数据导入报错

WITH READ LOCK / FOR EXPORT with pxc strict mode = ENFORCING

## Cause
- PXC does not support LOCK TABLES statements in strict mode ENFORCING

## Resolution
- Update pxc_strict_mode = DISABLE and restart service

## [workaround]
- Use PXC instance IP for data import instead of ProxySQL address
