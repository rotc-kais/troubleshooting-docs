# 应用界面查看事件报内部错误

应用管理界面查看事件报内部错误 错误信息: elastic: Error 400 (Bad Request): all shards failed [typessearch_phase_execution_exception] 跨多个命名空间复现

## Cause
- Elasticsearch索引及索引模板被误删除

## Resolution
- 清理Elasticsearch和Kafka数据重建索引
