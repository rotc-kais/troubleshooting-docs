# kafka咨询

kafka消费失败 用户dev-drm-kafka无对应的topic权限报错

## Cause
- 3.16.2版本Kafka资源(topic/group)命名禁止使用下划线导致权限配置失败

## Resolution
- 通过CR/YAML更新创建带下划线名称的topic
- 手动修改dev-drm-kafka用户的YAML添加topic和消费者组权限
- 删除历史错误配置项
- 建议升级到ACP 4.0版本
