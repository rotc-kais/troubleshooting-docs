# 3.8.1

ES集群内无audit索引生成 razor返回状态码500

## Cause
- Kafka接收最大传输包大小限制1M导致数据无法上传

## Resolution
- 单独升级razor镜像到3.8.8版本
- 最终在3.8.2版本解决
