# 日志存储插件部署失败

kafka服务启动失败退出 kafka事件服务健康检查失败

## Cause
- 数据清理不彻底导致残留数据

## Resolution
- 卸载日志存储插件
- 删除数据目录：/cpaas/data/zookeeper, /cpaas/data/kafka*, /cpaas/data/elasticsearch
- 重新安装日志存储插件
