# 3.8.2

业务集群部署日志存储插件（对接外部es）失败 lanaya组件运行异常 lanaya组件日志报错“Failed to parse mapping [_doc]”

## Cause
- 外部es版本为7.X，与当前平台版本3.10不兼容

## Resolution
- 将外部es版本更换为6.8.13
