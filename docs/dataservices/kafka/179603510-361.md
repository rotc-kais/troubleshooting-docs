# 3.6.1

apollo、lanaya、courier配置中的--kafka-topic-log参数填写不一致 外部ES中存在logstash-other-2023.10.16索引

## Cause
- lanaya组件配置的kafka-topic-log参数错误
- 存在第三方进程消费kafka数据写入ES

## Resolution
- 检查并修正lanaya组件的kafka-topic-log配置
- 排查是否存在第三方消费进程
