# 使用生产者消费者连接kafka失败

使用生产者消费者连接kafka失败

## Cause
- 创建kafkauser时authorization.type=group的name字段未配置正确值

## Resolution
- 修改KafkaUser资源的authorization配置，将group类型的name字段改为正确的值
