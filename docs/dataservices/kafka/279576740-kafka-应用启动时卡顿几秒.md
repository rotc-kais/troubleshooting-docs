# kafka 应用启动时卡顿几秒

kafka应用启动时卡顿几秒 Pod中有反向解析DNS请求，每次请求2秒后返回错误

## Cause
- 生产集群coredns ConfigMap缺少hosts配置导致DNS查询失败

## Resolution
- 在Pod中添加hosts配置临时解决
- 修改coredns ConfigMap添加hosts配置后删除Pod的hosts配置

## [workaround]
- 在Pod spec中配置hosts记录
