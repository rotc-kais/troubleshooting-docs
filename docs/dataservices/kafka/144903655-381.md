# 3.8.1

kafka实例日志报错部分文件内容因打开的文件太多导致无法加载 最大打开文档数量达到上限

## Cause
- segment.bytes参数设置为1048576k(1M)过小，频繁生成segment文件导致连接数耗尽

## Resolution
- 清理当前topic后重新创建，设置segment.bytes参数为100M
