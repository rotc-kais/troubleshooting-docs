# 配置kafka密码认证

## 环境信息 适用版本：3.6.x

## Cause

## Resolution
- 修改kafka-cr文件listeners字段为scram-sha-512
- 新建KafkaUser资源添加SCRAM-SHA-512认证
- 创建KafkaTopic资源定义分区和副本
- 验证步骤：检查Service/Secret状态、获取用户密码、配置客户端文件并测试生产消费
