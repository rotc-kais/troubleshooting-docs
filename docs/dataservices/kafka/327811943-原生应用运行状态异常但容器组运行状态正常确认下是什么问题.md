# 原生应用运行状态异常，但容器组运行状态正常，确认下是什么问题

原生应用状态显示异常/错误 实际容器组（Pod）运行状态正常

## Cause
- 原生应用在创建或更新时失败导致状态未同步

## Resolution
- 重新更新受影响的原生应用
