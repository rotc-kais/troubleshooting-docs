# 日志组件

cpaas-zookeeper组件启动失败 日志报错Unable to load database on disk

## Cause
- Zookeeper数据文件读取异常（version-2目录数据损坏或缺失）

## Resolution
- 在节点文件系统删除数据目录：rm -rf /cpaas/data/zookeeper/version-2
- 重启异常的cpaas-zookeeper服务Pod
