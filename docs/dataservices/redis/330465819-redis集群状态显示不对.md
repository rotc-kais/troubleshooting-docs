# redis集群状态显示不对

界面显示处理中，但实际redis已经启动成功 实例状态卡在初始化

## Cause
- 新建redis实例时使用了历史遗留的旧pvc

## Resolution
- 删除当前redis实例及关联的旧pvc
- 重新创建redis实例
