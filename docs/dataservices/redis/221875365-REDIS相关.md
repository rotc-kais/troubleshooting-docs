# REDIS相关

客户端连接超时 Redis日志出现"OOM"错误 主从同步延迟超过阈值

## Cause
- 内存不足触发OOM机制
- 网络延迟或带宽不足
- timeout/repl-timeout配置值过低

## Resolution
- 修改redis.conf: maxmemory 16gb && maxmemory-policy allkeys-lru
- 检查网络设备QoS配置
- 调整redis.conf: timeout 300 && repl-timeout 60

## [workaround]
- 临时增加maxmemory值或重启实例
- 设置客户端指数退避重试
