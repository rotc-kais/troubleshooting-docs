# redis卡在启动中

redis实例一直处在'处理中' rediscluster状态异常

## Cause
- 网络策略导致rediscluster组件间通信异常
- 平台升级到4.0版本后redis operator部署方式变更引发网络策略不兼容

## Resolution
- 检查并调整网络策略配置，确保redis组件间正常通信
- 完成平台4.0版本升级后重新部署redis operator
