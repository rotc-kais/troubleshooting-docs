# ace1.12环境opentsdb异常通过修复动作清理后，界面组件仍处于异常状态

界面组件仍处于异常状态 hbase-region显示处理中 postgres链接数达到上限（默认500，当前600+）

## Cause
- postgres链接数达到上限导致redis连接失败

## Resolution
- 重启postgres释放连接
- 重启redis组件
