# 微服务应用创建限流策略不生效

微服务应用创建限流策略后可以无限访问

## Cause
- 未部署 Redis 哨兵
- 集群维度配置中 Redis 配置未启用或配置错误

## Resolution
- 在网格配置-集群维度配置中配置 MasterName 为 mymaster
- 地址填写数据服务中已部署 Redis 的哨兵访问地址
