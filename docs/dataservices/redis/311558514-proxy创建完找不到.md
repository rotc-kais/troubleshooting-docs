# proxy创建完找不到

无法找到创建的Redis实例

## Cause
- operator未正确安装或配置
- CRD未正确注册
- 用户权限不足
- 资源创建过程出现错误

## Resolution
- 检查operator pod状态及日志
- 执行`kubectl get crd`确认Redis CRD
- 验证Redis CR的yaml配置
- 使用`kubectl get all -n <namespace>`查看资源
- 添加RBAC权限
