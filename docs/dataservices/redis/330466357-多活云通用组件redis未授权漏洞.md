# 多活云通用组件redis未授权漏洞

安全工具报告Redis存在未授权漏洞 网络抓包疑似发现未授权访问

## Cause
- 安全工具误判
- Redis已正确配置认证(requirepass/ACL)或网络访问控制

## Resolution
- 确认漏洞为误报，无需修复
- 检查redis.conf中requirepass配置
- 检查ACL访问控制列表
- 限制网络暴露范围
