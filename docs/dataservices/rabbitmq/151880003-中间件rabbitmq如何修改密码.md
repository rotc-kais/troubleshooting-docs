# 中间件rabbitmq如何修改密码

修改保密字典中的default-user配置后无法生效

## Cause
- RabbitMQ使用Erlang的Mnesia数据库存储密码，Secret仅用于首次密码初始化

## Resolution
- 使用kubectl进入每个实例执行命令：rabbitmqctl change_password Username 'Newpassword'
- 通过rabbitmqctl list_users验证用户列表
